HTMLCollection

HTMLCollection Object

Keywords Explaination Example
General array-like collection of HTML elements returned by:
.getElementsByTagName()
.getElementsByClassName()
.children
  • possible to loop through or access items by index
  • impossible to use array methods like valueOf(), pop(), push(), or join().
Inheritance not inherit

Methods & Properties

Keywords Explaination Example
length HTMLCollection.length
return number - the number of elements
useful to loop through an HTMLCollection

how many elements in a HTMLCollection

document.getElementsByTagName("p").length;
item() HTMLCollection.item(index)
HTMLCollection[index] simple
return element - the element by index

access the 1st element in a HTMLCollection

document.getElementsByTagName("p")[0];
namedItem() HTMLCollection.namedItem(name)
HTMLCollection[name] simple
return element - the element by name or id attribute

access the element with id "myFirstPara"

document.getElementsByTagName("p")["myFirstPara"];
