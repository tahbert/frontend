Trigonometry
Math.sin() return the sine (a value between -1 and 1) of the angle x (given in radians).
Math.cos() return the cosine (a value between -1 and 1) of the angle x (given in radians).
Math.log() return the natural logarithm of x
Math.log2() return the base 2 logarithm of x.
Math.log10() return the base 10 logarithm of x.
