Methods and Properties

Keywords Explaination Example
length

history.length

return number - the number of URLs in the history

get the number of URLs in the history

history.length;
go()

history.go(number)

load a URL (page) from the history
return none
  • 0: reload the page
  • negative: go back
  • positive: go forward

click the button to go back 2 pages

el.onclick = () => {
 history.go(-2);
}
forward()

history.forward()

load the next URL (page) in the history
return none
= history.go(1)

click the button to go to the next page

el.onclick = () => {
 history.forward();
}
back()

history.back()

load the previous URL (page) in the history
return none
= history.go(-1)

click the button to go to the prev page

el.onclick = () => {
 history.back();
}
