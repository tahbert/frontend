Arrow Function

Arrow Function

Elements Explaination Example
General () => instead of function()
useful shorter syntax
  • better to keep return and curly brackets
  • hoisting impossible, must be defined before being used
  • const is safer than var, a function exp is a constant value.

use regular function

const myFn1 = function() {return "Hi There!";}
myFn1();

use arrow function better

const myFn2 = () => {return "Hi There!";}
myFn2();

more simple if single statement

const myFn3 = () => "Hi There!";
myFn3();
with parameters

with parentheses better

myFn1 = (x) => "Hi " + x;
myFn1("There");

without parentheses if single statement

myFn2= x => "Hi " + x;
myFn2("There");
this keyword
  • regular func: refer to the object that calls the func
  • arrow func: refer to the owner of the func
  • not suitable for defining object methods
  • not suitable for call(), apply() and bind()
  • can't be used as a constructor
