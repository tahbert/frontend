Constructor

Elements Explaination Example
Object Constructor the function to create many objects of the same "type"
  • good practice name constructor with upper-case first letter
  • this keyword: no value, just a substitute for new objects
  • objects are created by calling constructor with new

1. create an object constructor

function Person(first, last, age) {
 this.fname = first;
 this.lname = last;
 this.age = age;
}
typeof Person;

2. create objects of the same type

const father = new Person("John", "Doe", 50, "blue");
const mother = new Person("Annie", "Sy", 48, "blue");

3. access object as normal

father.fname;
mother.fname;

Adding methods
 to an object => not be added to other objects

add a property to an object

father.nationality = "Vietnamese";

add a method to an object

father.name = function () {
 return this.fname + " " + this.lname;
}

access object as normal

father.nationality;
mother.nationality;
father.name();
mother.name();

done inside constructor function

function Person(first, last, age) {
 this.fname = first;
 this.lname = last;
 this.age = age;
 this.nationality = "Vietnamese" // add a property
 this.name = function () { // add a method
  return this.fname + " " + this.lname;
 }
}

access object as normal

father.nationality;
mother.nationality;
father.name();
mother.name();
