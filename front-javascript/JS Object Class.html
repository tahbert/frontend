Elements Explaination Example
Class class Name {
 constructor() {...}
}
template for objects of the same type
  • class keyword
  • class properties: use special method constructor() to initialize
  • class methods: same syntax as object methods
  • class must be written in "strict mode"
  • impossible hoisting: must declared before using

create a class

class Person {
 constructor(fname, lname) {
  this.fname = fname; // class property
  this.lname = lname; // class property
 }
 name() { // class method
  return this.fname + this.lname;
 }
 age(x) { // class method with params
  return x - this.birth;
 }
}
typeof Person;

const person1 = new Person("Tahbert", "Nguyen", 1994);
const person2 = new Person("Annie", "Sy", 1994);

Class Inheritance use extends keyword to create
use super() access the parent's properties and methods

 myCar.show();
Getters and Setters use get and set keywords
  • getter/setter name must differ from property name
    => many programmers use underscore with the same name
  • even if the getter/setter is a method, not use parentheses when getting/setting property value

get and then set

myCar.carname;

set and then get

myCar.carname;
