Function

Elements Explaination Example
Function block of code to perform a particular task
executed when "sth" invokes it (calls it).
    why useful
  • reusable: define once, use many times
  • different results: with different arguments each times
 function myFn(a, b) {return a*b;}
myFn(5, 2);
Function are Objects function has both properties and methods
=> best be described as objects
  • function as object property => object method
  • function to create new objects => object constructor
 function myFn(a, b) {return a*b;}

typeof (function myFn(a, b) {return a*b;});
Declaration
function name(param, param, etc) {
 // some code here
}
  • not a part of an executable statement => no semicolon
  • name: same rules as variables
 function myFn(a, b) {return a*b;}
myFn(5, 2);
can be declared by an expression, stored in a variable
after that the variable can be used as a function
  • part of an executable statement => end with semicolon
  • anonymous function (no name)
 const myFn = function (a, b) {return a*b};
myFn;
myFn(5, 2);
Hoisting possible to be called before declared
impossible when declared by an expression
myFn(5, 2) * 10;
function myFn(a, b) {return a*b}
Invocation
    3 ways:
  • automatically (self invoked)
  • when it is invoked (called) from JS code
  • when an event occurs (Ex. click a button)
Self-invoking when the expression is followed by () operator
anonymous self-invoking function (without name)

with normal declaration

(function() {return "Hi"})();

with normal declaration, arrow function

(() => {return "Hi"})();

declared by an expression

const x = function() {return "Hi"}();
x; executed => x; instead of x();

declared by an expression, arrow function

const y = (() => {return "Hi"})();
y;
Function Sequence funcs are executed in the sequence they are called, not declared function displayHi() {

return keyword when return statement reached
=> stop executing and returning value to the caller
() operator
    when calling a fuction
  • without (): refer to the function object
  • with (): refer to the function result
 myFn;
