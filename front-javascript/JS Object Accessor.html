Elements Explaination Example
General get and set allow to define Object Accessors
  • simpler syntax
  • allow equal syntax for properties and methods
  • can secure better data quality
  • useful for doing things behind-the-scenes
get keyword get the value of an property
(instead of using method as property)

get the value of the language property

const x = {
 name: "Tahbert",
 language: "en",
 get lang() {
  return this.language;
 }
};
x.lang;
set keyword set the value of an property
(instead of using method as property)

set the value of the language property

const x = {
 name: "Tahbert",
 language: "",
 set lang(value) {
  this.language = value;
 }
};
x.lang = "en";
x.language;
