DOMTokenList

DOMTokenList Object

Keywords Explaination Example
General a set of space-separated tokens returned by: classList
  • possible to loop through and access items by index
  • impossible to use array methods like valueOf(), pop(), push(), or join().

return all the element's classnames

el.classList;
Inheritance not inherit
