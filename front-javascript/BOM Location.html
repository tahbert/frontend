Methods & Properties

Keywords Explaination Example
Heading
origin

location.origin

return string - protocol, hostname, port number of a URL.
port number default or not specified => empty

starting code:

host: www.w3schools.com

get the origin of the URL

el.origin; // result: https://www.w3schools.com
hash

location.hash

location.hash = name

return or set string - the anchor part of the URL, including #

starting code:

<a id="demo" href="tahbert.com#tech"></a>
const el = document.getElementById("demo");

get the anchor part of the URL:

el.hash; // result: #tech

set the anchor part of the URL:

el.hash = "music"; // result: #music
host

location.host

location.host = host:port

set or return string - the host (IP address or domain) and port number of a URL.
port number default or not specified => empty

starting code:

host: www.w3schools.com

get the host of the URL:

el.host; // result: www.w3schools.com
hostname

location.hostname

location.hostname = name

set or return string - the host (IP address or domain) of a URL

starting code:

host: www.w3schools.com

get the hostname of the URL:

el.hostname; // result: www.w3schools.com
href

location.href

location.href = URL

set or return string - the full page URL (include the protocol like https://)

starting code:

host: www.w3schools.com

get the href of the URL

el.href; // result: https://www.w3schools.com
pathname

location.pathname

location.pathname = name

return string - the current pathname

starting code:

host: www.w3schools.com/tech.asp

get the pathname of the URL

el.pathname; // result: tech.asp
port

location.port

location.port = port

set or return string - the port number of a URL
port number default or not specified => empty

starting code:

host: www.w3schools.com

get the port of the URL

el.port; // result empty
protocol

location.protocol

location.protocol = protocol

return string - the protocol of the current URL, including the colon (:)

starting code:

host: www.w3schools.com

get the protocol of the URL

el.protocol; // result: https:
search

location.search

location.search = querystring

return string - the querystring part of a URL, including the question mark (?)

starting code:

host: www.w3schools.com/?answer=yes

get the search of the URL

el.search; // result: ?answer=yes
assign()

location.assign()

loads a new document
return none

load a new document

location.assign("https://www.w3schools.com");
replace()

location.replace()

replace the current document with a new one
return none
remove the current URL from the document history
=> impossible to navigate back to the original document.

replace a document

location.replace("https://www.w3schools.com");

location.reload()

reload the current document
return none

reload the document

location.reload();
