Object Set

Elements Explaination Example
Set a collection of unique values
Sets are objects
  • each value can only occur once
  • value can be of any data type
 const x = new Set(["a","b","c"]);
typeof x;
Properties
size return number - the number elements in a Set const x = new Set(["a","b","c"]);
x.size;
Methods > Creating
new Set() pass an array to the new Set() constructor const x = new Set(["a","b","c"]);
add() create a new Set and use add() to add values // create a Set
const x = new Set();
// add values to the Set
x.add("a");
x.add("b");
x.add("c");
create a new Set and use add() to add variables // create a Set
const x = new Set();
// create variables
const a = "a";
const b = "b";
const c = "c";
// add variables to the Set
x.add(a);
x.add(b);
x.add(c);
Methods > Other
forEach() set.forEach()
call a function (callback) for each element
 const x = new Set(["a","b","c"]);
text;
values() set.values()
return object - new iterator object with values		 const x = new Set(["a","b","c"]);
x.values();
text;
keys() set.keys()
return the same as values()
this makes Sets compatible with Maps
 const x = new Set(["a","b","c"]);
x.keys();
text;
entries() set.entries()
return value,value pairs instead of key,value pairs
this makes Sets compatible with Maps
 const x = new Set(["a","b","c"]);
x.entries();
text;
