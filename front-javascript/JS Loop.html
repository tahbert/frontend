Label continue labelname;
break labelname;
  • continue without label: skip 1 loop iteration
  • continue with label: skip 1 loop iteration
  • break without label: jump out of loop or switch
  • break with label: jump out of any code block

break with lable to break a block scope

const arr = ["Ford", "BMW", "Tesla"];
myLable: {
 text += arr[0] + " ";
 text += arr[1] + " ";
 break myLable;
 text += arr[2] + " ";
}
text;
