NamedNodeMap

NamedNodeMap

Keywords Explaination Example
NamedNodeMap collection of Attr objects
  • look like an array but not, accessed by name or index
  • unlike NodeList, objects inside a NamedNodeMap are not in any particular order
Inheritance not inherite any method or propety

Methods & Properties

Keywords Explaination Example
length namednodemap.length
return number - the number of attribute nodes in a NamedNodeMap

return how many attributes the element has

el.attributes.length;
item() namednodemap.item(index)
namednodemap.[index]
return node - the attribute node at the specified index

return the first attribute

el.attributes[0];
getNamedItem() namednodemap.getNamedItem("name")
return node - an attribute node from a NamedNodeMap.
It is easier to use the el.getAttribute()

get the value of "src" attribute

el.attributes.getNamedItem("src").value;
setNamedItem() namednodemap.setNamedItem("name")
add an attribute node (by name)
return node - the replaced node (if any).
it's easier to use el.setAttribute()
already exist => replace, otherwise the return value is null.

create and set "src" attribute

const nodeMap = el.attributes;
const node = document.createAttribute("src");
node.value = "avatar.jpg";
nodeMap.setNamedItem(node);
removeNamedItem() namednodemap.removeNamedItem("name")
remove a node (by name)
return node - the removed node
remove input type => type="text" as default

remove the "type" attribute of an input

el.attributes.removeNamedItem("type")
