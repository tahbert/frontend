Properties and Methods

Keywords Explaination Example
Getting Elements
querySelector() el.querySelector(selectors)
return element - the first child element matched

single: get the 1st element with "class1"

el.querySelector(".class1");

multi: get the 1st element with "class1" or "class2"

el.querySelector(".class1, .class2");
querySelectorAll() el.querySelectorAll(selectors)
return NodeLlist - all child elements matched

single: get all elements with "box1"

el.querySelectorAll(".box1");

single: get all elements with "box1" and "green"

el.querySelectorAll(".box1.green");

multi: get all elements with "box1" and all elements with "box2"

el.querySelectorAll(".box1, .box2");
getElementsByClassName()

el.getElementsByClassName(class)

return HTMLCollection - all child elements with specified class(es)

get all child elements with class box

el.getElementsByClassName("box");

get all child elements with class box and green

el.getElementsByClassName("box green");
getElementsByTagName() el.getElementsByTagName(tagname)
return HTMLCollection - all child elements with a specified tagname.

get all child element of el with tagname p

el.getElementsByTagName("p");

get the 1st child element of el with tagname p

el.getElementsByTagName("p")[0];

get all child element of el

el.getElementsByTagName("*");

get the 1st child element of el

el.getElementsByTagName("*")[0];

return boolean - true if matched (multi: comma-separeted)

Attribute
attributes el.attributes
return NamedNodeMap - all the element's attributes
getAttribute() el.getAttribute("name")
return string - the value of attribute with a specified name
getAttribute("name") = getAttributeNode("name").value

get value of the attribute "display"

el.getAttribute("display");
getAttributeNode() el.getAttributeNode("name")
return Attr - the attribute with a specified name
getAttributeNode("name").value = getAttribute("name")

get the attribute "display"

el.getAttributeNode("display");

get value of the attribute "display"

el.getAttributeNode("display").value;
setAttribute() el.setAttribute("name", "value")
return none - set a new value to an attribute
attribute not exist => positioned first
add value to a style attribute, it can cause overwrite.

add an "class" attribute

el.setAttribute("class", "demo");
setAttributeNode() el.setAttributeNode(node)
add an attribute to an element (replace the existing if nany)
return Attr - the replaced attribute node if any
1. create an attribute object
2. set the attribute value
3. add the attribute to an element
=> setAttribute() is more easier

add an "class" attribute

const attr = document.createAttribute("class");
attr.value = "demo";
el.setAttributeNode(attr);
removeAttribute() el.removeAttribute("name")
remove an attribute from an element
return none

remove the attribute "class"

el.removeAttribute("class");
removeAttributeNode() el.removeAttributeNode(node)
remove an attribute from an element
return Attr - the removed attribute
1. create an attribute object
2. add the attribute to an element
=> removeAttribute() is more easier

remove the attribute "class"

const attr = element.getAttributeNode("class");
el.removeAttributeNode(attr);
hasAttribute() el.hasAttribute("name")
return boolean - true if the element has the specified attribute

check if the element have attribute "class"

el.hasAttribute("class");

if attribute "class" exists, change value to "demo"

if (el.hasAttribute("class")) {
 el.setAttribute("class", "demo");
}
hasAttributes() el.hasAttributes()
return boolean - true if the element has attributes

check if the element have attributes

el.hasAttributes();

Insert | Remove | Replace
remove() el.remove()
remove an element from the document
return none
  • the element is removed from the DOM.
  • return none => can't be modified and inserted back into the DOM.

remove an element from the document

el.remove();
insertAdjacentElement() el.insertAdjacentElement(position, element)
insert an element into a specified position.
position param: afterbegin| afterend | beforebegin| beforeend

Basic code for examples:

const h2 = document.getElementById("myH2");
const span = document.getElementById("mySpan");

move <span> to after </h2> tag:

h2.insertAdjacentElement("afterend", span);

move <span> to before </h2> tag:

h2.insertAdjacentElement("beforeend", span);
insertAdjacentHTML() el.insertAdjacentHTML(position, html)
insert HTML code into a specified position
position param: afterbegin| afterend | beforebegin| beforeend

Basic code for examples:

const h2 = document.getElementById("myH2");
const span = "<span>my new span.</span>";

insert 1 <span> after </h2> tag

h2.insertAdjacentHTML("afterend", span);

insert 1 more <span> after </h2> tag

h2.insertAdjacentHTML("afterend", span);
insertAdjacentText() el.insertAdjacentText(position, text)
insert a Text into a specified position
position param: afterbegin| afterend | beforebegin| beforeend

Basic code for examples:

const h2 = document.getElementById("myH2");
let text = "some text";

insert text after <h2> tag

h2.insertAdjacentText("afterbegin", text);

insert text before </h2> tag

h2.insertAdjacentText("beforeend", text);
later replaceChildren(), replaceWith()
Return in pixels
clientHeight el.clientHeight
return number - viewable height, including padding, not border, scrollbar, margin.

box-sizing: content-box; height: 100px; padding: 10px; border: 5px;

el.clientHeight; result: 120px

box-sizing: border-box; height: 100px; padding: 10px; border: 5px;

el.clientHeight; result: 90px
clientWidth el.clientWidth
return number - viewable width, including padding, not border, scrollbar, margin.

box-sizing: content-box; width: 100px; padding: 10px; border: 5px;

el.clientWidth; result: 120px

box-sizing: border-box; width: 100px; padding: 10px; border: 5px;

el.clientWidth; result: 90px
clientLeft el.clientLeft
return number - the left border width.
the same as style.borderLeftWidth

border: 10px 5px;

el.clientLeft; result: 5px
clientTop el.clientTop
return number - the top border width.
the same as style.borderTopWidth

border: 10px 5px;

el.clientTop; result: 10px

return DOMRect - the element's size and its position relative to the viewport (left, top, right, bottom, x, y, width, height) 		example
Return in pixels > Scroll
scrollIntoView() el.scrollIntoView(true|false)
return none - scroll an element into the visible area
  • true - the element top aligned to the visible area top
  • false - the element bottom aligned to the the visible area bottom
depending on the layout of other elements, may not be scrolled completely

scroll to the top of the element

el.scrollIntoView();

scroll to the bottom of the element

el.scrollIntoView(false);
scrollHeight el.scrollHeight
return number - the height of an element
(including padding, but not borders, scrollbars, margins)
return the entire height of an element including what is not viewable (because of overflow).
example
scrollWidth el.scrollWidth
return number - the width of an element
(including padding, but not borders, scrollbars, margins)
return the entire width of an element including what is not viewable (because of overflow).
example
scrollLeft el.scrollLeft
el.scrollLeft = pixels
set or return number - how far the element has been scrolled horizontally
  • negative => set to 0.
  • can't be scrolled => set to 0.
  • greater than maximum => set to the maximum.

return how far the element is scrolled from the left

el.scrollLeft;

set how far the element is scrolled from the left

el.scrollLeft = 50;

set how far each time the element is scrolled from the left

el.scrollLeft += 50;
scrollTop el.scrollTop
el.scrollTop = pixels
return or set number - how far the element has been scrolled vertically
  • negative => set to 0.
  • can't be scrolled => set to 0.
  • greater than maximum => set to the maximum.

return how far the element is scrolled from the top

el.scrollTop;

set how far the element is scrolled from the top

el.scrollTop = 50;

set how far each time the element is scrolled from the top

el.scrollTop += 50;

Interface: FocusEvent
blur when an element has lost focus
most often used with form validation
cancelable | bubble
=> using with useCapture = onfocusout

el.addEventListener("focus", myFocus, true);
el.addEventListener("blur", myBlur, true);

function myFocus() {
 console.log("Input has received focus");
}
function myBlur() {
 console.log("Input has lost focus");
}
focus when an element has received focus
most often used with <input>, <select>, <a>
cancelable | bubble
=> using with useCapture = onfocusin
focusin when an element is about to get focus
using to determine whether an element or its child gets focus
may not work in Chrome, Safari with JS HTML DOM syntax

Interface: MouseEvent
General

order in DOM for the left/middle button:

  • onmousedown
  • onmouseup
  • onclick

order in DOM for the right button

  • onmousedown
  • onmouseup
  • oncontextmenu
mousedown when a user presses a mouse button over an element el.onmousedown = (event) => {
 event.target.style.color = "green";
};
mouseup when the user releases a mouse button over an element. el.onmouseup = (event) => {
 event.target.style.color = "grey";
};
click when the user clicks on an element.
fire after 1 pair of mousedown and mouseup
el.onclick = (event) => {
 event.target.style.color = "red";
};
dblclick when the user double-clicks on an element.
fire after 2 pairs of mousedown and mouseup
el.ondblclick = (event) => {
 event.target.style.color = "red";
};
contextmenu when the user right-clicks on an element to open the context menu
contextmenu event is supported in all browsers
contextmenu attribute is currently only supported in Firefox.
el.oncontextmenu = (event) => {
 event.target.style.color = "red";
};
