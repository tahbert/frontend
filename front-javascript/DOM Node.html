Methods & Properties

Keywords Explaination Example
Relationship
parentNode node.parentNode
returnn node - the parent node
  • return null if no parent node available
  • more popular than parentElement

access to the parent element

el.parentNode;
parentElement node.parentElement
return element - the parent element
  • return null if no parent node available
      or available but NOT an element node
  • less popular than parentNode

access to the parent element

el.parentElement;
childNodes node.childNodes
returnn NodeList - all elements's child nodes
childNodes[0] is the same as firstChild
return nodes: element nodes, text nodes, comment nodes.
whitespace between elements are also text nodes.

access to the 1st child node

el.childNodes[0];
firstChild node.firstChild
return node - the first child node of a node.
the same as childNodes[0]
return node: element node, text node, and comment node.
whitespace between elements are also text nodes.

access to the 1st child node

el.firstChild;
lastChild node.lastChild
return node - the last child node of a node.
return node: element node, text node, and comment node.
whitespace between elements are also text nodes.

access to the last child node

el.lastChild;
previousSibling node.previousSibling
return node - the previous node on the same tree level
return node: element node, text node, and comment node.
whitespace between elements are also text nodes.

access to the prev sibling node

el.previousSibling;
nextSibling node.nextSibling
return node - the next node on the same tree level.
return node: element node, text node, and comment node.
whitespace between elements are also text nodes.

access to the next sibling node

el.nextSibling;
hasChildNodes() node.hasChildNodes()
return boolean - true if the node has any child nodes
whitespace between nodes are considered child nodes (text nodes).

check if the element has any child nodes

el.hasChildNodes();
contains() node.contains(node)
return boolean - true if a node is a descendant of aother node

check if the element contains another element

el1.contains(el2);
isEqualNode() node.isEqualNode(node)
return boolean - true if two nodes are equal
    equal if qualifying all conditions:
  • same nodeType
  • same nodeName
  • same nodeValue
  • same nameSpaceURI
  • same childNodes with all the descendants
  • same attributes and attribute values
  • same localName and prefix

check if two elements are equal

el1.isEqualNode(el2);
isSameNode() node.isSameNode(node)
return boolean - true if the two nodes are the same.
by Firefox => use "==="

check if two elements are the same

el1.isSameNode(el2);
nodeName node.nodeName
return string - the node name

return the node name

el.nodeName;
nodeType node.nodeType
return number - the node type
  • 1: element node
  • 2: attribute node
  • 3: text node
  • 8: comment node
  • 9: document node

return the node type

el.nodeType; // result: 1
nodeValue node.nodeValue
node.nodeValue = value
set or return string - the node value

change text inside the element

el.childNodes[0].nodeValue = "changed";
Insert | Remove | Replace
appendChild() node.appendChild(node)
return node - append a node as the last child

create an "li" node:

const node = document.createElement("li");

create a text node:

const textnode = document.createTextNode("Water");

append the text node to the "li" node:

node.appendChild(textnode);

append the "li" node to the list:

document.getElementById("myList").appendChild(node);
insertBefore() node.insertBefore(newnode, existingnode)
return node - insert a node before an existing child

create a "li" element:

const newNode = document.createElement("li");

create a text node:

const textNode = document.createTextNode("Water");

append text node to "li" element:

newNode.appendChild(textNode);

insert before the first child:

const list = document.getElementById("myList");
list.insertBefore(newNode, list.children[0]);
cloneNode() node.cloneNode(deep)
return node - the clone node
    deep prama: optional
  • false - clone only the node and its attributes
  • true - clone the node, its attributes, and its descendants

copy the last child of myList2 to myList1

const node = document.getElementById("myList2").lastChild;
const clone = node.cloneNode(true);
document.getElementById("myList1").appendChild(clone);
removeChild() node.removeChild(node)
return node - remove a child node.
  • the child node is removed from the DOM
  • return node => can be modified and inserted back into the DOM.

remove the last element in myList

const list = document.getElementById("myList");
const last = list.lastElementChild;
list.removeChild(last);
replaceChild() node.replaceChild(newnode, oldnode)
return node - replace a child node with a new node

replace 1st child element text of myList

const el = document.getElementById("myList").children[0];
const textNode = document.createTextNode("Water");
el.replaceChild(textNode, el.childNodes[0]);
normalize() node.normalize()
remove empty text nodes, and join adjacent text nodes.
useful to remove unnecessary existing nodes
 complex
Return Text
textContent node.textContent
node.textContent = text
set or return string - the node text content (and its descendants)
including spacing, CSS hidden text, without tags
setting will replace all child nodes by 1 text node

change the element text content

el.textContent = "changed!";
Others
ownerDocument node.ownerDocument
return Document - the owner document of a node
In HTML, always the HTML document itself

return nodeType of the owner document

el.ownerDocument.nodeType; // result: 9
compareDocumentPosition() node.compareDocumentPosition(node)
return number - indicate where they are positioned in the document
  • 1: two nodes do not belong to the same document
  • 2: the 1st is after the 2nd
  • 4: the 1st is before the 2nd
  • 8: the 1st is inside the 2nd
  • 16: the 2nd is inside the 1st
  • 20: the 1st is before the 2nd and the 2nd is inside the 1st
  • 32: two nodes are attributes on the same element
  • return value can also be a combination of values.

compare positions of two elements

el1.compareDocumentPosition(el2);
