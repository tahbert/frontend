HTMLSelectElement

HTMLSelectElement Object

Keywords Explaination Example
General the object representing a <select> element
Inheritance EventTarget <= Node <= Element <= HTMLElement <= HTMLSelectElement

Methods and Properties

Keywords Explaination Example
autofocus

HTMLSelectElement.autofocus

HTMLSelectElement.autofocus = "true|false"

set or return boolean - reflect autofocus attribute

check autofocus attribute

selectEl.autofocus;
disabled

HTMLSelectElement.disabled

HTMLSelectElement.disabled = "true|false"

set or return boolean - reflect disabled attribute

check disabled attribute

selectEl.disabled;
form

HTMLSelectElement.form

return HTMLFormElement - the associated <form> element

return the associated form

selectEl.form;
labels

HTMLSelectElement.labels

return NodeList - all associated <lable> elements

return the associated labels

selectEl.labels;
options

HTMLSelectElement.options

return HTMLOptionsCollection - all associated <option> elements

return the associated options

selectEl.options;
selectedIndex

HTMLSelectElement.selectedIndex

return number - indicate the selected <option> element
no selected <option> => -1

assume that the fist option is seleted

selectEl.selectedIndex; // result: 0
selectedOptions

HTMLSelectElement.selectedOptions

return HTMLCollection - all seleted <option> elements (many if multiple attribute exists)

return all seleted <option>

selectEl.selectedOptions;
type

HTMLSelectElement.type

return string - form control's type
  • select-one: one value can be selected
  • select-multiple: multiple values can be selected

assume that only one value can be selected

selectEl.type; // result: "select-one"
item()

HTMLSelectElement.item(index)

HTMLSelectElement[index]

return element - the <option> by index

access the 1st <option>

selectEl[0];
namedItem()

HTMLSelectElement.namedItem(id|name)

return element - the <option> by name or id attribute

access the <option> with id "opt1"

selectEl.namedItem("opt1");
add()

HTMLSelectElement.add(item)

HTMLSelectElement.add(item, beforeitem)

add an <option> or <optgroup>
return none

add an <option> as a last child of <select>

selectEl.add(optionEl);

add an <option> before the first child of <select>

selectEl.add(optionEl, selectEl.options[0]);
remove()

HTMLSelectElement.remove(index)

remove an <option> by index
return none

remove the first <option> of <select>

selectEl.remove(0);
checkValidity()

HTMLSelectElement.checkValidity()

check whether the element has any constraints and whether it satisfies them
return none 		later
setCustomValidity()

HTMLSelectElement.setCustomValidity(message)

set the custom validity message for the element
return none 		later
