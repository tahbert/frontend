Storage

Storage Object

Keywords Explaination Example
localStorage store key/value pairs in the browser
the data is not deleted when the browser is closed
sessionStorage store key/value pairs in the browser
the data is deleted when the browser is closed

Methods and Properties

Keywords Explaination Example
key()

localStorage.key(index)

sessionStorage.key(index)

return string - the item key by index

return the 1st item key of localStorage

localStorage.key(0);
length

localStorage.length

sessionStorage.length

return number - the number of items stored

how many items stored in localStorage

localStorage.length;
setItem()

localStorage.setItem(key, value)

sessionStorage.setItem(key, value)

set the value of the specified Storage Object item
return string - the inserted value

set Items for localStorage

localStorage.setItem("fname", "Tahbert");
localStorage.setItem("lname", "Nguyen");
getItem()

localStorage.getItem(key)

sessionStorage.getItem(key)

return string - the value of the specified Storage Object item

get Items from localStorage

localStorage.getItem("fname");
localStorage.getItem("lname");
removeItem()

localStorage.removeItem(key)

sessionStorage.removeItem(key)

remove an Storage Object item by key
return none

remove an Item from localStorage

localStorage.removeItem("lname");
clear()

localStorage.clear()

sessionStorage.clear()

remove all the Storage Object item for this domain
return none

remove all localStorage item

localStorage.clear();
