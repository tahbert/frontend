Elements Explaination Visual Code
General Set of rules, how JS programs are constructed // How to create variables:
var x;
let y;
// How to use variables:
x = 5;
y = 6;
let z = x + y;
Value Literal: fixed value
Numbers are written with or without decimals
10.50
1001
Strings are written within double or single quotes:
"John Doe"
'John Doe'
Variable: variable value
  • variables are used to store data values
  • keywords var, let, const are used to declare variables
  • equal sign is used to assign values to variables
Operator arithmetic operator: to compute values( + - * / ) (5 + 6) * 10;
assignment operator: to assign values to variables x = 5;
Expression A combination of values, variables, and operators, which computes to a value. 5 * 10
"John" + " " + "Doe"
Identifiers/Names Used to name variables, keywords, functions
First character: letter/ underscore (_)/ dollar sign ($)
Subsequent characters: letters/ digits/ underscores/ dollar signs
 		let lastname;
let _lastname;
let $lastname;
Case sensitive let lastname, lastName;
lastName = "Doe";
lastname = "Peterson";
Different ways of joining multiple words into one variable name
Lower Camel Case tendency
firstName, lastName, masterCard, interCity. 		Upper Camel Case (Pascal Case): allowed
FirstName, LastName, MasterCard, InterCity.
Underscore: allowed
first_name, last_name, master_card, inter_city. 		Hyphens: banned
first-name, last-name, master-card, inter-city.
