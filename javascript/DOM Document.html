Document

Document Object

Keywords Explaination Example
General Document object: when a web page is loaded
HTMLdocument object: when an HTML document is loaded (to distinguish from XMLDocument)
  • the owner of all other objects in an HTML document
  • the root node of an HTML document
  • a property of the window object
Inheritance EventTarget <= Node <= Document

Methods & Properties

Keywords Explaination Example
Getting Elements
getElementById() document.getElementById("id")
return element - the element by id
many elements with the same id => return the first

return the element with the id "demo"

document.getElementById("demo");
querySelector() document.querySelector("selector, selector, etc")
return element - the 1st element matched

single: get the 1st element with "class1"

document.querySelector(".class1");

multi: get the 1st element with "class1" or "class2"

document.querySelector(".class1, .class2");
querySelectorAll() document.querySelectorAll("selector, selector, etc")
return NodeLlist - all elements matched

single: get all elements with "box1"

document.querySelectorAll(".box1");

single: get all elements with "box1" and "green"

document.querySelectorAll(".box1.green");

multi: get all elements with "box1" and all elements with "box2"

document.querySelectorAll(".box1, .box2");
getElementsByClassName() document.getElementsByClassName("class class etc")
return HTMLCollection - all elements by classname(s).

single: get all elements with class "box"

document.getElementsByClassName("box");

multi: get all elements with both class "box" and "green"

document.getElementsByClassName("box green");
getElementsByTagName() document.getElementsByTagName("tagname")
return HTMLCollection - all elements by tagname.

get all elements with tagname "p"

document.getElementsByTagName("p");

get all elements in the document

document.getElementsByTagName("*")
getElementsByName() document.getElementsByName(name)
return live NodeList - all elements by name
  • IE works in a different way
  • IE, Edge return HTMLCollection

get all elements with the name "animal"

document.getElementsByName("animal");
Creating Elements
createElement() document.createElement("tagName")
create HTMLElement - the HTMLelement by tagName

1. create an element

const para = document.createElement("p");

2. set content for the element

para.innerText = "Hi There.";

3. add the element to the body

document.body.appendChild(para);
createDocumentFragment() document.createDocumentFragment()
  • create a node (an offscreen node that can be used to build a new document fragment that can be insert into any document)
  • can also be used to extract parts of a document, change, add, or delete some of the content, and insert it back to the document.
  • why don't edit elements directly? Inserting the fragment when it is ready, there will be only one reflow and one single render.
  • useful when appending elements in a loops, it improves performance.
document nodes appended to the document fragment, are removed from the original document.
example
adoptNode() document.adoptNode(node)
adopt a node from another document.
  • descendants nodes are also adopte.
  • original node and descendants is removed

adopt the 1st <h1> from a frame

const frame = document.getElementById("myFrame");
const h1 = frame.contentWindow.document.getElementsByTagName("H1")[0];
const node = document.adoptNode(h1);
document.body.appendChild(node);
importNode() document.importNode(node, deep)
import a node from another document
    deep param: required
  • true: descendant nodes are also imported.
  • false: only the node itself is imported.
original node is NOT removed

import the 1st <h1> from a frame

const frame = document.getElementById("myFrame");
const h1 = frame.contentWindow.document.getElementsByTagName("H1")[0];
const node = document.importNode(h1, true);
document.body.appendChild(node);
Creating Other Nodes
createAttribute() document.createAttribute("name")
create node - an attribute node
return node - the created attribute node
setAttribute() is easier to use

1. create an attribute:

const att = document.createAttribute("class");

2. set the value of the attribute:

att.value = "democlass";

3. add the attribute to an element:

el.setAttributeNode(att);
createTextNode() document.createTextNode("text")
create node - a text node
return node - the created text node
can be used to escape HTML characters

1. create a text:

const text = document.createTextNode("Hi There");

2. add the text to an element:

el.appendChild(text);
createComment() document.createComment("text")
create node - a comment node
return node - the created comment node

1. create an comment:

const cmt = document.createComment("scripts here");

2. add the comment to the body:

document.body.appendChild(cmt);
createEvent() document.createEvent("type")
create object - an event object
return object - the created event object
  • the event must be of a legal event type
  • the event must be initialized (dipatched) before use
many methods used with this method are deprecated

Write, Edit, Modify
lastModified document.lastModified
return string - the date and time the document was last modified.

alert the date and time the document was last modified

alert(document.lastModified);

transform lastModified into a Date object

const lastModified = new Date(document.lastModified);
designMode document.designMode
document.designMode = "on|off"
return or set if the document is editable.

check if the document is editable

document.designMode;

set the iframe editable

const frame = document.getElementById("myFrame");
frame.contentDocument.designMode = "on";
write() document.write(exp1, exp2, ...)
write directly to an open (HTML) document stream
most often used to write to output streams opened by open()
  • deletes all existing HTML when used on a loaded document
  • can NOT be used in XHTML or XML
writeln() document.writeln(exp1, exp2, ...)
write directly to an open (HTML) document stream
write a newline character (U+000A) after each statement
=> It makes no sense to use writeln(), just use <br>
  • deletes all existing HTML when used on a loaded document
  • can NOT be used in XHTML or XML
open() document.open()
open a document for writing
return none
all existing document content will be cleared.
document.open() is called automatically when
document.write() is called after the page has loaded.
close() document.close()
close a window previously opened with open().
return none
clear the document (overwrite all previous content)
Basic Information
documentURI document.documentURI
document.documentURI= locationURI
set or return string - the document location (full URL)
can be used on any document types.

return the document location:

document.documentURI;
URL document.URL
return string - the document location (full URL)
can only be used on HTML documents

return the document location:

document.URL;
location document.location
return object - containing information about the URL of the document and provides methods for changing that URL and loading another URL.

print a location object to the console

console.log(document.location);
referrer document.referrer
return string - URL of the document that loaded the current document.
return empty string - if the document was not opened with a link
inside an <iframe>, document.referrer will initially be set to the same value as the href of the parent window's window.location

return the referrer of the document:

document.referrer;

return string - the loading status of the current document

Keywords Explaination Example
Interface: Event
scroll when the document has been scrolled document.onscroll = () => {
 alert("the document has been scrolled");
};
fullscreenchange when the browser switches into or out of fullscreen mode.
sent to the Element, then bubbles up to the Document
prefixes | cancelable
requestFullscreen() to open fullscreen mode
exitFullscreen() to exit fullscreen mode.

standard

document.addEventListener("fullscreenchange", function() {
 alert("fullscreenchange event fired!");
});

for Firefox

document.addEventListener("mozfullscreenchange", function() {
 alert("mozfullscreenchange event fired!");
});

for Chrome, Safari and Opera

document.addEventListener("webkitfullscreenchange", function() {
 alert("webkitfullscreenchange event fired!");
});

for IE, Edge

document.addEventListener("msfullscreenchange", function() {
 alert("msfullscreenchange event fired!");
});
fullscreenerror when the browser can NOT switch to fullscreen mode.
two events fired: the 1st sent the Element, the 2nd sent to the Document
prefixes | cancelable

standard

document.addEventListener("fullscreenerror", function() {
 alert("Fullscreen denied!");
});

for Firefox

document.addEventListener("mozfullscreenerror", function() {
 alert("Fullscreen denied!");
});

for Chrome, Safari and Opera

document.addEventListener("webkitfullscreenerror", function() {
 alert("Fullscreen denied!");
});

for IE, Edge

document.addEventListener("msfullscreenerror", function() {
 alert("Fullscreen denied!");
});
Interface: DragEvent > Fired on the draggable target
dragstart when the user starts to drag an element or text selection
cancelable
document.addEventListener("dragstart", (event) => {
 event.target.classList.add("dragging");
});
drag when an element or text selection is being dragged
while dragging, fires every 350 milliseconds.
 document.addEventListener("drag", (event) => {
 console.log("dragging");
});
dragend when the user has finished dragging an element or text selection
cancelable
document.addEventListener("dragend", (event) => {
 event.target.classList.remove("dragging");
});
Interface: DragEvent > Fired on the drop target
dragenter when a dragged element or text selection enters a valid drop target.
about to enter a drop target.
 complex
dragover when an element or text selection is being dragged over a valid drop target
fires every 350 milliseconds.
can't be dropped by default => use preventDefault()
 complex
dragleave when a dragged element or text selection leaves a valid drop target
about to leave a drop target.
 complex
drop when an element or text selection is dropped on a valid drop target complex
