Number Method
toString() Return a number as a string let toString1 = 123;
toString1.toString(); // result: from variable x
(123).toString(); // result: from literal 123
(100 + 23).toString(); // result: from expression 100 + 23
valueOf() Return a number as a number let valueOf1 = 123;
valueOf1.valueOf(); // result: from variable x
(123).valueOf(); // result: from literal 123
(100 + 23).valueOf(); // result: from expression 100 + 23
toExponential() Return a string, with a number rounded and written using exponential notation
Parameter is specified => not round the number
 let toExponential= 9.656;
toExponential.toExponential(2); // result:
toExponential.toExponential(4); // result:
toExponential.toExponential(6); // result:
