Arithmetic perform an arithmetic operation on numbers
  • operation: (Ex: 2 + 5;)
  • operand: (Ex: 2, 5)
  • operator: (Ex: +)
Arithmetic Operator Example
Addition + let x = 5 + 2;
Subtraction - let x = 5 - 5;
Multiplication * let x = 5 * 2;
Division / let x = 5 / 2;
Modulus % let x = 5 % 2; // result: 1
Increment ++ let x = 5; x++; // result: 6
Decrement -- let x = 5; x--; // result: 4
Exponentiation ** let x = 5 ** 2; // result: 25
// the same as Math.pow(x,y)
let x = Math.pow(5,2);

Comparison compare values or variables
Operator Description Example
== equal value let x = 5;
x == 5; // true
x == "5"; // true
x == 8; // false
=== equal value and equal type let x = 5;
x === 5; // true
x === "5"; // false
x === "8"; // false
!= not equal value let x = 5;
x != 5; // false
x != "5"; // false
x != 8; // true
!== not equal value and equal type let x = 5;
x !== 5; // false
x !== "5"; // true
x !== 8; // true
> greater than let x = 5; x > 8; // false
< less than let x = 5; x < 8; // true
>= greater than or equal value to let x = 5;
x >= 8; // false
x >= 5; // true
<= less than or equal value to let x = 5;
x <= 8; // true
x <= 5; // true
Logical determine the logic between variables or values
Operator Description Example
&& and let x = 5; x < 10 && x < 1; // false
|| or let x = 5; x < 10 || x < 1; // true
! not let x = 5; !(x > 10); // false
