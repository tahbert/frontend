Nested Function All functions have access to the global scope
In fact, in JS, all functions have access to the scope "above" them.
=> Nested functions have access to the scope "above" them.

 let counter = 0;
 function plus() {counter += 1;}
 plus();
 return counter;
}
// the counter is now 1

Arrow Function
Elements Explaination Visual Code
General () => instead of function()
Why useful: to write a shorter function
  • Better to keep return keyword and curly brackets
  • Not hoisted, must be defined before being used
  • const is safer than var, because a function expression is always constant value.
// Using regular function
af1 = function() {
return "Hello World!";
}
af1(); // result:
// Using arrow function better
af2 = () => {

af2(); // result:
// More simple if one statement
af3 = () => "Hello World!";
