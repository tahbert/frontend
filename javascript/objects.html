General In JS, almost "everything" is an object:
  • Dates|Maths|Regular expressions|Arrays|Functions|Objects are always objects
  • Booleans|Numbers|Strings can be objects (if defined with the new keyword)
Declaring common practice declare with the const keyword. const person = {firstName:"John", lastName:"Doe", age:50};
Variables Objects are variables that can contain many values. const person = {firstName:"John", lastName:"Doe", age:50};
Properties and Methods: JS objects are containers for properties and methods.

The values can be: primitive values, other objects, functions

General A Map holds key-value pairs where the keys can be any datatype.
A Map remembers the original insertion order of the keys.
A Map has a property that represents the size of the map.
 const om1 = new Map([
 ["apples", 500],
 ["bananas", 300],
 ["oranges", 200]
]);
typeof om1; // result:
Compare to Object
Object Map
Iterable Not directly iterable Directly iterable
Size Not have a size property Have a size property
Key Types Keys must be Strings (or Symbols) Keys can be any datatype
Key Order Keys are not well ordered Keys are ordered by insertion
Defaults Have default keys Not have default keys
