Number + String
 let x = 16 + "Volvo"; // result: 16Volvo

Dynamic The same variable can be used to hold different data types let x; // Now x is undefined
x = 5; // Now x is a Number
x = "John"; // Now x is a String
String In single or double quotes let carName1 = "Volvo XC60"; // In double quotes

Number With, or without decimals let x1 = 34.00; // with decimals
let x2 = 34; // without decimals
Extra large or small numbers with exponential notation let y = 123e5; // 12300000
let z = 123e-5; // 0.00123

(x == 3); // false

Object Variables that can contain many property:value pairs const person = {firstName:"John"};
// property: firstName | value: John
const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
// 4 properties: firstName, lastName, age, and eyeColor
How to access object property:
  1. objectName.propertyName
  2. objectName["propertyName"]
 person.lastName;
person["lastName"];
Object method: functions that can be performed on objects, stored as a property const person = {
  firstName: "John",
  lastName: "Doe",
  id: 5566,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};
How to access object method (an object property)
  1. objectName.methodName
 person.fullName();

typeof "John" // Returns "string"
typeof (3 + 4) // Returns "number"
