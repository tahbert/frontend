Elements Explaination Example
Date Object the object working with dates and times
JS convert Date to string to display
instead of displaying [object Date]
const x = new Date();
x;
typeof x;
Date Formats
Date Input "yyyy-mm-dd"
ISO date (standard, JS prefered)
Date and time is separated with a capital T
UTC time is defined with a capital letter Z

year, month, day

const x = new Date("2015-03-25");
x;

year, month

const y = new Date("2015-03");
y;

year only

const z = new Date("2015");
z;
"mm/dd/yyyy"
short date 		const x = new Date("03/25/2015");
x;
"mmm dd yyyy"
"dd mmm yyyy"
long date (most often)

month first day after

const x = new Date("Mar 25 2015");
x;

day first month after

const y = new Date("25 Mar 2015");
y;

month can be in full name

const z = new Date("March 25 2015");
z;
Date Output independent of input, always in full text string
 Wed Mar 25 2015 07:00:00 GMT+0700 (Indochina Time)
Time Zones the browser's time zone by default

Methods > Creating
new Date() new Date()
create a new date object with the current date and time 		const x = new Date();
x;
new Date(year, month)
with a specified year, month
JS counts months from 0 to 11
year with 1 or 2 digit will be interpreted as 19xx:
const x = new Date(2018, 11);
x;
const y = new Date(99, 11,);
y;
const z = new Date(9, 11,);
z;
new Date(year, month, day)
with a specified year, month, day
 const x = new Date(2018, 11, 24);
x;
new Date(year, month, day, hour)
with a specified year, month, day, hour
 const x = new Date(2018, 11, 24, 10);
x;
new Date(year, month, day, hour, minute)
with a specified year, month, day, hour, minute
 const x = new Date(2018, 11, 24, 10, 33);
x;
new Date(year, month, day, hour, minute, second)
with a specified year, month, day, hour, minute, second
 const x = new Date(2018, 11, 24, 10, 33, 30);
x;
new Date(year, month, day, hour, minute, second, millisecond)
with a specified year, month, day, hour, minute, second, millisecond
 const x = new Date(2018, 11, 24, 10, 33, 30, 0);
x;
new Date(string)
create a new date object from a date string 		const x = new Date("October 13, 2014 11:13:00");
x;
new Date(milliseconds)
create a new date object as zero time plus milliseconds
start at January 1, 1970, 00:00:00 Universal Time (UTC)
const x = new Date(0);
x;
const y = new Date(100000000000);
y;
