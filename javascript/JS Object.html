Object

Elements Explaination Example
General variable that can contain many values
    almost "everything" is an object
  • object is object
  • array, function, regexp is object
  • Date, Math is object
  • numbers, boolean, string is object if declared with new
Declaring common practice with const const x = {fname:"Tahbert", lname:"Nguyen", age:28};
Creating declare and assign properties at the same time const x = {fname:"Tahbert", lname:"Nguyen", age:28};
declare and assign properties later const x = {};
x.fname = "Tahbert";
x.lname = "Nguyen";
x.age = 28;
