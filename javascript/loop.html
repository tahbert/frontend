For Loop
Elements Explaination Visual Code
General Loop through a block of code a number of times
for (statement 1; statement 2; statement 3) {
  // code block
}
 // result:
Statement 1 Multiple values => separated by comma
let i1, lo3Len, lo3Text;
for(i1=0, lo3Len=lo3.length, lo3Text="<ul>" ; i1<lo3Len; i1++){...}
 // result:
Can omitted (like setting values before the loop)
let i2 = 0, lo4Len = lo4.length, lo4Text = "<ul>";
for(; i2<lo4Len; i2++){...}
 // result:
Statement 2 Can omitted => break statement inside the loop required
Statement 3 Can omitted => like increasing values inside the loop
for(let i=0; i<lo5Len;) {
  lo5Text += "<li>" + lo5[i] + "</li>";
  i++;
}
 // result:
Negative increment (i--), positive increment (i = i + 15), or anything else.
for(let i=lo6Len-1; i>=0; i--) {
  lo6Text += "<li>" + lo6[i] + "</li>";
}
 // result:
Loop Scope Using var in a loop var i = 5;
for (var i = 0; i < 10; i++) {...}
// Here i = 10
Using let in a loop let i = 5;
for (let i = 0; i < 10; i++) {...}
// Here i = 5

Labels
continue statement without a label skip one loop iteration
continue statement with a label
continue labelname; 		skip one loop iteration
break statement without a label jump out of a loop or a switch
break statement with a label
break labelname; 		can be used to jump out of any code block
A code block is a block of code between { and } 		// result:
