Switch Perform different actions based on different conditions
switch(expression) {
  case x:
  // code block
  break;
  case y:
  // code block
  break;
  default:
  // code block
}
 break keyword: break out of the switch block
(not necessary to break the last case)
default keyword: specify the code to run if there is no case match
(not have to be the last case in a switch block)
