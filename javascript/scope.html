Scope
Elements Explaination Visual Code
General Scope determines the accessibility of variables
Lifetime
    Lifetime of variables:
  • Function/Local variables: Start when declared and deleted when the function is completed.
  • Global variables: Start when declared and deleted when the browser or tab is closed.
Block Scope Variables declared inside a { } block
(With var keyword => no use)
{
 const x = 2;
}
// x can NOT be used here
{
 let x = 2;
}
// x can NOT be used here
{
 var x = 2;
}
// x CAN be used here
Function|Local Scope Variables defined inside a function
Not accessible from outside the function.
(=> Variables with the same name in different functions is possible)
// carName can NOT be used here function myFunction() {
 const carName = "Volvo";
 // carName CAN be used here
}
// carName can NOT be used here
// carName can NOT be used here function myFunction() {
 let carName = "Volvo";
 // carName CAN be used here
}
// carName can NOT be used here
// carName can NOT be used here function myFunction() {
 var carName = "Volvo";
 // carName CAN be used here
}
// carName can NOT be used here
Global scope Variables declared outside a function
Accessible from anywhere in a JS program.
warning: Not create global variables unless you intend to
 const carName = "Volvo";
// carName CAN be used here function myFunction() {
 // carName CAN be used here
}
// carName CAN be used here
 let carName = "Volvo";
// carName CAN be used here function myFunction() {
 // carName CAN be used here
}
// carName CAN be used here
 var carName = "Volvo";
// carName CAN be used here function myFunction() {
 // carName CAN be used here
}
// carName CAN be used here
