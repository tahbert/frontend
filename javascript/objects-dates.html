Creating
new Date() Create a new date object with the current date and time const do1 = new Date();
do1; // result:
new Date(year, month, ...) Create a new date object with a specified date and time.
Note: JS counts months from 0 to 11
7 parameters: year, month, day, hour, minute, second, millisecond const do2 = new Date(2018, 11, 24, 10, 33, 30, 0);
do2; // result:
6 parameters: year, month, day, hour, minute, second const do3 = new Date(2018, 11, 24, 10, 33, 30);
do3; // result:
5 parameters: year, month, day, hour, minute const do4 = new Date(2018, 11, 24, 10, 33);
do4; // result:
4 parameters: year, month, day, hour const do5 = new Date(2018, 11, 24, 10);
do5; // result:
3 parameters: year, month, day const do6 = new Date(2018, 11, 24);
do6; // result:
2 parameters: year, month const do7 = new Date(2018, 11);
do7; // result:
1 and 2 digit years will be interpreted as 19xx: const do8 = new Date(99, 11,);
do8; // result:
const do9 = new Date(9, 11,);
do9; // result:
new Date(dateString) Create a new date object from a date string const do10 = new Date("October 13, 2014 11:13:00");
do10; // result:
new Date(milliseconds) Create a new date object as zero time plus milliseconds
Start: January 1, 1970, 00:00:00 Universal Time (UTC) 		const do11 = new Date(0);
do11; // result:
const do12 = new Date(100000000000);
do12; // result:
Displaying Dates
toString() Output dates in full text string format
Automatically converted to a string, with the toString() method 		const do13 = new Date();
do13; // result:
// the same as using toString()
const do14 = new Date();
do14.toString(); // result:
toUTCString() Convert a date to a UTC string (standard) const do15 = new Date();
do15.toUTCString(); // result:
toISOString() Convert a Date object to a ISO string (standard) const do16 = new Date();
do16.toISOString(); // result:
toDateString() Convert a date to a more readable format const do17 = new Date();
do17.toDateString(); // result:
Date Formats
Date Input|Output
Date Input
Type Example
ISO Date "2015-03-25" (International Standard)
Short Date "03/25/2015"
Long Date "Mar 25 2015" or "25 Mar 2015"
Date Output
Independent of input format, JS outputs dates in full text string format:
Ex: Wed Mar 25 2015 07:00:00 GMT+0700 (Indochina Time)
ISO Dates The preferred JS date format const do18 = new Date("2015-03-25");
do18; // result:
ISO Dates (Year and Month) const do19 = new Date("2015-03");
do19; // result:
ISO Dates (Year only) const do20 = new Date("2015");
do20; // result:
ISO Dates (Date-Time)
Date and time is separated with a capital T
UTC time is defined with a capital letter Z
 		const do21 = new Date("2015-03-25T12:00:00Z");
do21; // result:

// Relative to UTC, remove the Z and add +HH:MM or -HH:MM instead:
const do22 = new Date("2015-03-25T12:00:00-06:30");
do22; // result:
Short Dates "MM/DD/YYYY" syntax const d = new Date("03/25/2015");
Long Dates "MMM DD YYYY" syntax (most often) // Month first or Day first
const d = new Date("Mar 25 2015");
const d = new Date("25 Mar 2015");
// month can be in full or abbreviated
const d = new Date("January 25 2015");
