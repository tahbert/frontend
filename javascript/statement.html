General If a computer program is a list of instructions
=> a JS program is a list of programming statements.
  • Composed of: values, operators, expressions, keywords, comments.
  • Executed one by one, in the same order as they are written.
Within statements
Semicolons ( ; ) Separate JS statements (not required, but highly recommended) let x = y + z;
White Space Spaces are ignored => add white space to make it more readable let x = y + z;
Long Code The best place to break it is after an operator document.getElementById("demo").innerHTML =
"Hello Dolly!";
Code Block {...} Define statements to be executed together. function myFunction() {
  document.getElementById("demo1").innerHTML = "Hello Dolly!";
  document.getElementById("demo2").innerHTML = "How are you?";
}
Keyword Started to identify the JS action to be performed
(reserved words => cannot be used as names for variables, labels, or function names)
 		let x = y + z;
