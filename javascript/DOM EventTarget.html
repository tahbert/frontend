addEventListener() EventTarget.addEventListener(event, function, useCapture);
attach an event handler without overwriting existing event handlers
    recommended way to register an event listener
  • allows adding more than one handler for an event
  • finer-grained control of the phase when the listener is activated (capturing vs. bubbling) in comparation to onXYZ property
  • work on any event target, not just HTML or SVG elements.

internal function

el.addEventListener("click", function(){ alert("Hi!"); });

external function

el.addEventListener("click", myFunction);

passing parameters

using an "anonymous function" that calls the specified function with the parameters: 		el.addEventListener("click", function() {myFunction(p1, p2);});

useCapture: event propagation

    define events order
  • false: bubbling - inner to outer
  • true: capturing - outer to inner

bubbling as default

el.addEventListener("click", myFunction);

capturing when set to true

el.addEventListener("click", myFunction, true);
see example
removeEventListener() EventTarget.removeEventListener(event, function, useCapture);
remove event handlers that have been attached
removal of a capturing listener does not affect a non-capturing version of the same listener, and vice versa. If a listener is registered twice, one with the capture flag set and one without, => must remove each one separately
