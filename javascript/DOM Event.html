Event

Keywords Explaination Example
General when an event occurs, it belongs to a certain event object
(Ex, a mouse click event belongs to the MouseEvent object.)
Inheritance Event
Properties and Methods
target event.target
return element - the element that triggered the event
the element on which the event originally occurred
not necessarily the eventlistener's element.

return differently based on the element clicked in <body>

document.body.onclick = (e) => {
 alert(e.target.tagName);
}
currentTarget event.currentTarget
return element - the element whose event listeners triggered the event
particularly useful during capturing and bubbling.

return the same when clicking on any elements in <body>

document.body.onclick = (e) => {
 alert(e.currentTarget.tagName);
} return: BODY
type event.type
return string - the type of the triggered event.

return the type of triggered event

document.body.onclick = (e) => {
 alert(e.type);
} return: click
timeStamp event.timeStamp
return number - the number of milliseconds from the document was finished loading until the specific event was created.
not all systems provide this information

return how long I have visited the web

document.body.onclick = (e) => {
 alert(e.timeStamp);
}
bubbles event.bubbles
returns boolean - true if an event is a bubbling one.

check if the onclick event is a bubbling one

document.body.onclick = (e) => {
 alert(e.bubbles);
} return: true
cancelable event.cancelable
return boolean - true if the event is a cancelable one.
cancelable if it's possible to prevent default

check if the onclick event is a cancelable one

document.body.onclick = (e) => {
 alert(e.cancelable);
} return: true
preventDefault() event.preventDefault()
cancel the event if cancelable
returns none
stopPropagation() to prevent further propagation of an event through the DOM

cancel onclick event on the frist link

document.links[0].onclick = (event) => {
 event.preventDefault();
};
defaultPrevented event.defaultPrevented
reutrn boolean - true if preventDefault() was called for the event.

check onclick event on the frist link

document.links[0].onclick = (event) => {
 event.preventDefault();
 alert(event.defaultPrevented);
}; return: true
stopPropagation() event.stopPropagation()
prevent propagation of the same event from being called
returns none
propagation: bubbling up to parent elements or capturing down to child elements.
complex
stopImmediatePropagation() event.stopImmediatePropagation()
prevent other listeners of the same event from being called.
returns none 		complex
composedPath() event.composedPath()
return array - the elements in the event flow, in execution order.
by Edge
 complex
isTrusted event.isTrusted
reutrn boolean - true if the event is trusted
by Safari
  • Chrome, Firefox, Opera: trusted if invoked by the user, not if by a script
  • IE: all are trusted except those created with createEvent()
 complex
eventPhase event.eventPhase
reutrn number - which phase of the event flow is currently being evaluated. 		later
later composed

UiEvent

Keywords Explaination Example
General Events that are triggered from the user interface.
Inheritance Event <= UiEvent
Properties and Methods
detail event.detail
return number - the number of clicks in a short time
  • onclick: current click count.
  • onmousedown|onmouseup: current click count + 1
  • ondblclick: always "2"
  • onmouseover|onmouseout: always "0"

return the number of double click

el.ondblclick = (event) => {
 event.detail;
}; return: 2
view event.view
return object - the Window object where the event occured.

return the Window object where the event occured.

el.onclick = (event) => {
 event.view;
}; return: [object Window]

MouseEvent

Keywords Explaination Example
General Events that occur when the mouse interacts with the HTML document
Inheritance Event <= UiEvent <= MouseEvent
Properties and Methods
button event.button
returns number - which mouse button was pressed
when a mouse event was triggered.
  • 0: left mouse
  • 1: middle button
  • 2: right button
mostly used with onmousedown

return which mouse button is pressed

el.onmousedown = (event) => {
 alert(event.button);
}
which event.which
return number - which mouse button was pressed
when a mouse event was triggered.
result the same as button + 1
mostly used with onmousedown

return which mouse button is pressed

el.onmousedown = (event) => {
 alert(event.which);
}
buttons event.buttons
returns number - which mouse button/buttons was pressed
when a mouse event was triggered.
  • 1: left button
  • 2: right button
  • 4: middle button
  • 8: fourth button (typically "Browser Back")
  • 16: fifth button (typically "Browser Forward")
mostly used with onmousedown
by Safari

return which mouse button/buttons is pressed

el.onmousedown = (event) => {
 alert(event.buttons);
}

Keywords Explaination Example
General Events that occur when the mouse wheel is scrolling
Do NOT confuse the wheel event with the scroll event.
Inheritance Event <= UiEvent <= MouseEvent <= WheelEvent
Properties and Methods
deltaMode event.deltaMode
returns number - the length unit of the scrolling values.
  • 0: pixels
  • 1: lines
  • 2: pages
by Safari

return the length unit of the scrolling values

document.body.onwheel = (event) => {
 alert(event.deltaMode);
}
deltaX event.deltaX
returns double -
  • positive: when scrolling to the right
  • negative: when scrolling to the left
  • 0: otherwise
by Safari
most mouse devices have no ability to scroll along the z-axis

return deltaX

document.body.onwheel = (event) => {
 alert(event.deltaX);
}
deltaY event.deltaY
returns double
  • positive: when scrolling down
  • negative: when scrolling up
  • 0: otherwise
by Safari

return deltaY

document.body.onwheel = (event) => {
 alert(event.deltaY);
}
deltaZ event.deltaZ
returns double -
  • positive: when scrolling in
  • negative: when scrolling out
  • 0: otherwise
by Safari
most mouse devices have no ability to scroll along the z-axis

return deltaZ

document.body.onwheel = (event) => {
 alert(event.deltaZ);
}
