Math Object

General
Elements Explaination Visual Code
General Allow us to perform mathematical tasks on numbers
Unlike other objects, the Math object has no constructor.
The Math object is static
 		Math.PI; // result:
Creating All methods and properties can be used without creating a Math object first

Trigonometry
Math.sin(x) Return the sine (a value between -1 and 1) of the angle x (given in radians).
Math.cos(x) Return the cosine (a value between -1 and 1) of the angle x (given in radians).
Math.log(x) Return the natural logarithm of x
Math.log2(x) Return the base 2 logarithm of x.
Math.log10(x) Return the base 10 logarithm of x.
