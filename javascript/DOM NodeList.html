NodeList

NodeList Object

Keywords Explaination Example
General array-like collection of nodes returned by:
.getElementsByName()
.querySelectorAll()
.childNodes
  • possible to loop through or access items by index
  • impossible to use array methods like valueOf(), pop(), push(), or join().
HTMLCollection
or NodeList

Similarity

  • array-like object
  • length property

Difference

  • HTMLCollection items can be accessed by their name, id
  • NodeList can contain attribute nodes and text nodes

Methods & Properties

Keywords Explaination Example
length NodeList.length
return number - the number of nodes
useful for loop

how many nodes in a NodeList

const nodeList = document.querySelectorAll(".box");
nodeList.length;
