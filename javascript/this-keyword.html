This Keyword

This Keyword
Elements Explaination Visual Code
General Refer to the object it belongs to
Alone Refer to the global object let x = this; // refer to [object Window]

"use strict";
let x = this; // refer to [object Window]
Within Method Refer to the owner object. // refer to person object
const person = {
 firstName: "John",
 lastName : "Doe",
 id : 5566,
 fullName : function() {
 return this.firstName + " " + this.lastName;
 }
};
Explicit Function Binding
Methods like call(), and apply() can refer this to any object. 		// Refer to person2, even if it is a method of person1:
const person1 = {
 fullName: function() {
 return this.firstName + " " + this.lastName;
 }
}
const person2 = {
 firstName:"John",
 lastName: "Doe",
}
person1.fullName.call(person2); // return "John Doe"
Within Function Refer to the global object function myFunction() {
 return this; // refer to [object Window]
}

"use strict";
function myFunction() {
 return this; // undefined
}
