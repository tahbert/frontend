String method: return a new string, not modify the original string
slice() Extract a part of a string and return in a new string
start from 0
Positive parameter: left to right
let sliceEx = "Apple, Banana, Kiwi";
sliceEx.slice(7, 13) // result:
Negative parameter: right to left
let sliceEx = "Apple, Banana, Kiwi";
sliceEx.slice(-12, -6) // result:
2nd parameter omitted => slice out the rest
let sliceEx = "Apple, Banana, Kiwi";
sliceEx.slice(7) // result:
substring() Similar to slice(), but not accept negative indexes.
Positive parameter: left to right
let substringEx = "Apple, Banana, Kiwi";
substringEx.substring(7, 13) // result:
2nd parameter omitted => slice out the rest
let substringEx = "Apple, Banana, Kiwi";
substringEx.substring(7) // result:
substr() Similar to slice(), but the 2nd parameter specifies the length of the extracted part
Positive parameter: left to right
let substrEx = "Apple, Banana, Kiwi";
substrEx.substr(7, 6) // result:
2nd parameter omitted => slice out the rest
let substrEx = "Apple, Banana, Kiwi";
substrEx.substr(7) // result:

String search method
indexOf() string.indexOf(searchvalue, start)
Return the index of the first occurrence of a specified text
Return -1 if the text is not found
Can't take powerful search values (regular expressions)) like search()
 		let indexOfEx = "How are you today? Are you ok?";
// Find the 1st occurrence of "you"
indexOfEx.indexOf("you"); // result:
// Find the 1st occurrence of "you" (start at index 15)
indexOfEx.indexOf("you", 15); // result:
// Find the 1st occurrence of "me"
indexOfEx.indexOf("me"); // result:
lastIndexOf() Return the index of the last occurrence of a specified text
Return -1 if the text is not found
Can't take powerful search values (regular expressions)) like search()
 		let lastIndexOfEx = "How are you today? Are you ok?";
// Find the 1st occurrence of "you" (from end to start)
lastIndexOfEx.lastIndexOf("you"); // result:
// Find the 1st occurrence of "you" (start at index 15, from end to start)
lastIndexOfEx.lastIndexOf("you", 15); // result:
// Find the 1st occurrence of "me"
lastIndexOfEx.lastIndexOf("me"); // result:

match() string.match(regexp)
Search for a match against a regular expression, and return the matches, as an Array object
 let matchEx = "The rain in SPAIN stays mainly in the plain";
// Without g modifier (global search), return the first match
matchEx.match(/ain/); // result:
// With g modifier, return the array
matchEx.match(/ain/g); // result:
// With gi - global search and case-insensitive
matchEx.match(/ain/gi); // result:
// Using a string "ain", not regexp
matchEx.match("ain"); // result:

Automatic replacing of variables with real values is called string interpolation

Expression Substitutions: allow expressions in strings let price = 10;
