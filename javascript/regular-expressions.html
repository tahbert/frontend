General /pattern/modifier(s);
A sequence of characters that forms a search pattern, make your search much more powerful
 A case-insensitive pattern
let pattern = /w3schools/i;
Modifiers (Flags )
Keywords Explaination Visual Code
i Insensitive: perform a case insensitive match (ignores case of [a-zA-Z]) let m1 = "Is this all there is?";
m1.match(/is/i); // Result:
g Global: perform a global match, not only the first match
 let m2 = "Is this all there is?";
m2.match(/is/g); // Result:
m2.match(/is/gi); // Result:
m Multiline: cause ^ and $ to match the begin/end of each line (not only begin/end of string)
    Affect the behavior of start ^ and end $ only:
  • ^ specifies a match at the start of each line.
  • $ specifies a match at the end of each line.
 let m3 =
`Is this
all there
is`;
 m3.match(/^is/gim); // Result:
m3.match(/is$/gim); // Result:
m3.match(/^is$/gim); // Result:
u Unicode: pattern strings are treated as UTF-16. Also causes escape sequences to match unicode characters
Patterns
Keywords Explaination Visual Code
Matching Symbol
. Match any character
tahbert Match a block of characters: tahbert
^tahbert Match the block tahbert at the begining of each line
tahbert$ Match the block tahbert at the end of each line
\btahbert\b Boundary: Match the word tahbert
Meta Character
Metacharacters Characters with a special meaning: \d Find a digi
let r13 = "Give 100%!";
r13.match(/\d/g); // Result:
\s Find a whitespace character
let r14 = "This is Tahbert Nguyen";
r14.match(/\s/g); // Result:
\b - Find a match at the beginning of a word like this: \bWORD
let r15 = "This is Tahbert Nguyen";
r15.search(/\bis/); // Result:
\b - Find a match at the end of a word like this: WORD\b
r15.search(/is\b/); // Result:
\uxxxx - Find the Unicode character specified by the hexadecimal number xxxx
let r16 = "Visit W3Schools. Hello World!";
r16.match(/\u0057/g); // Result:
Quantifier
Quantifiers Define quantities n+ - Matches any string that contains at least one n
let r17 = "Hellooo World! Hello W3Schools!";
r17.match(/o+/g); // Result:
n? - Matches any string that contains zero or one occurrences of n
let r19 = "1, 100 or 1000?";
r19.match(/10?/g); // Result:
n* - Matches any string that contains zero or more occurrences of n
let r18 = "Hellooo World! Hello W3Schools!";
r18.match(/lo*/g); // Result:
Range and Group
Range [] A range of alternative characters [abc] - Find any of the characters between the brackets
let r10 = "Is this all there is?";
r10.match(/[hs]/g); // Result:
[0-9] - Find any of the digits between the brackets
let r11 = "123456789";
r11.match(/[1-4]/g); // Result:
Group () Alternative blocks of characters (x|y) Find any of the alternatives separated with |
let r12 = "re, green, red, green, gren, gr, blue, yellow";
r12.match(/(red|green)/g); // Result:
() can contain []
Extended
Keywords Explaination Visual Code
search() With a string let re1 = "My name is Tahbert Nguyen";
// Do a search for "W3schools" in a string:
re1.search("Tahbert"); // Result:
With a regular expression let re1 = "My name is Tahbert Nguyen";
// Do a case-insensitive search for "w3schools" in a string:
re1.search(/tahbert/i); // Result:
replace() With a string let re3 = "My name is Tahbert Nguyen";
// Replace a specified value with another value
re3.replace("Tahbert", "Duong"); // Result:
With a regular expression let re4 = "My name is Tahbert Nguyen";
// Use a case insensitive regular expression to replace
re4.replace(/tahbert/i, "Duong"); // Result:
test() Search a string for a pattern, and return true or false /e/.test("The best things in life are free!"); // Result:
exec() Search a string for a pattern, and return the found text as an object. /e/.exec("The best things in life are free!"); // Result:
