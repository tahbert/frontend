var keyword
Asigning Assign a value when declaring var carName = "Volvo";
Not assign a value when declaring var carName;

multiple declaration var person = "John Doe", carName = "Volvo", price = 200;
Redeclaration allowed Not lose its value var x = 2; // Now x is 2
var x = 3; // Now x is 3
Block Scope Can be accessed from outside the block {
  var x = 2;
}
// x CAN be used here
Arithmetic var x = 5 + 2 + 3;
String Strings will be concatenated: var x = "John" + " " + "Doe";
// If a number is in quotes, the rest will be treated as strings:
var x = "5" + 2 + 3; // result: 523
var x = 2 + 3 + "5" + 2; // result : 552
// strings and number => return a string:
var x = "Hello" + 5; // result : Hello5

Const keyword
Elements Explaination Visual Code
When to use Always declare a variable with const keyword unless you know that the value will change
=> Use it when declare a new array, object, function, regExp
Objects and Arrays const keyword doesn't define a constant value, but defines a constant reference to a value
    => you CAN:
  • Change the elements of constant array
  • Change the properties of constant object
 const cars = ["Saab", "Volvo", "BMW"]; // Creating a constant array:
cars[0] = "Toyota"; // You can change an element:
cars.push("Audi"); // You can add an element:
const car = {type:"Fiat", model:"500", color:"white"}; // Creating a const object:
car.color = "red"; // You can change a property:
car.owner = "Johnson"; // You can add a property:
    => you can NOT:
  • Reassign a constant value
  • Reassign a constant array
  • Reassign a constant object
 const car = "Saab";
car = "Volvo"; error

const cars = ["Saab", "Volvo", "BMW"];
cars = ["Toyota", "Volvo", "Audi"]; error

const car = {type:"Fiat", model:"500", color:"white"};
car = {type:"Volvo", model:"EX60", color:"red"}; error
