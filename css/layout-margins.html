Margins

Elements Explaination Visual Code
margin margin: length|auto|initial|inherit; (default: 0)
Set space around elements, outside of any defined borders
Negative values not allowed
More Specific margin-top, margin-right, margin-bottom, margin-left
Mixing Properties According to clock direction
margin: 5px; 4 margins
margin: 5px 10px; top + bottom | right + left
margin: 5px 10px 15px; top | right + left | bottom
margin: 5px 10px 15px 20px; top | right | bottom | left
Margin Collapse Sometimes two margins collapse into a single margin.
Happen on top and bottom margins
margin: 0 0 50px 0; margin-bottom: 50px
margin: 20px 0 0 0; margin-top: 20px has ben collapsed => total 50px
