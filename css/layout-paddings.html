Paddings

Keywords Explaination Visual Code
padding padding: length|initial|inherit; (default: 0)
Specify space around an element's content, inside of any defined borders
Negative values not allowed
More Specific padding-top, padding-right, padding-bottom, padding-left
Mixing Properties According to clock direction
padding: 5px; 4 paddings
padding: 5px 10px; top + bottom | right + left
padding: 5px 10px 15px; top | right + left | bottom
padding: 5px 10px 15px 20px; top | right | bottom | left
Padding and Width Problem: If an element has a specified width
=> the padding will be added to the total width of the element

Solution: Using box-sizing: content-box; to mantain the width
=>When increasing the padding, the available content space will decrease see more
