CSS variable
  • Make the code easier to read (more understandable)
  • Make it much easier to change the color values
Global variable: Can be accessed/used through the entire document, declared inside the :root selector :root {
  --main-bg-color: coral;
  --blue: #0000ff;
}
Local variable: Used only inside the selector where it is declared button {
  --main-bg-color: coral;
  --blue: #0000ff;
}
var() function var(--name, value)
Used to insert the value of a CSS variable
  • name required the variable name (must start with two dashes)
  • value optional the fallback value if the variable is not found
 #div {
  color: var(--blue);
  background-color: var(--main-bg-color);
}
Overriding Variables Specify local variable to override Global Variable :root {
  --blue: #1e90ff;
  --white: #ffffff;
}

body {
  background-color: var(--blue); <!-- #1e90ff -->
}

button {
  --blue: #0000ff;
  border: 1px solid var(--blue); <!-- #0000ff -->
}
Specify local variable to be used in that local element button {
--button-blue: #0000ff;
border: 1px solid var(--button-blue); <!-- #0000ff -->
}
