Counters

Keywords Explaination Example
General CSS counters are like "variables". The variable values can be incremented by CSS rules
(which will track how many times they are used)

This is a default counter

counter-reset: my-section; (= my-section 0;)

counter-increment: my-section; (= my-section 1;)

content: "Section " counter(my-section) ": ";
counter-reset counter-reset: none|id number|initial|inherit;
Create or reset one or more counter
id: define which counter to reset
number: set the value the counter is reset to on each occurrence of the selector (default 0)

This changes counter-reset

counter-reset: my-section 1;

counter-increment: my-section; (= my-section 1;)

content: "Section " counter(my-section) ": ";
counter-increment counter-increment: none|id number|initial|inherit;
Increase or decrease the value of one or more CSS counters
id: define which counter to increment
number: set how much the counter will increment on each occurrence of the selector
(default 1, Negative values are allowed)

This changes counter-increment

counter-reset: my-section;

counter-increment: my-section 2;

content: "Section " counter(my-section) ": ";
content content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit;
Used with the ::before and ::after, to insert generated content
counter() or counters() counter(id, counterstyle)
Add the value of a counter to an element
id: required the same as id of counter-reset and counter-increment
counterstyle: can be a list-style-type value
