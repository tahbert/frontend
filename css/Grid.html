grid-row shorthand grid-row: grid-row-start/grid-row-end; (default: auto/auto)
define on which row(s) to place an item
grid-column shorthand grid-column: grid-column-start/grid-column-end; (default: auto / auto)
define on which column(s) to place an item
grid-row-start grid-row-start: auto|row-line specify on which row-line the item will start
grid-column-start grid-column-start: auto|span n|column-line specify on which column-line the item will start, or the number of columns the item will span
grid-row-end grid-row-end: auto|span n|row-line specify on which row-line the item will end, or how many rows an item will span
grid-column-end grid-column-end: auto|span n|column-line specify on which column-line the item will end, or how many columns an item will span
