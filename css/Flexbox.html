flex-grow flex-grow: number|initial|inherit; (default: 0)
specify how much an item grow relative other items to take up space
  • matter of how much an item takes up avaialbe space
  • not matter of its size when the window expands
  • overide flex-basis or width if specified (horizontally)
  • overide flex-basis or height if specified (vertically)
