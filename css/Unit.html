General
  • omitted => 0 as default

Absolute not recommended for use on screen
but can be used if the output medium is known, such as for print layout
Unit Explanation
px pixels
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)
Relative scale better between different rendering mediums
  • em, rem practical in creating perfectly scalable layout
  • viewport the browser window size
Unit Explanation
rem relative to the root element font-size
em relative to the current element font-size
% relative to the parent element
vw relative to 1% of the width of the viewport
vh relative to 1% of the height of the viewport*
vmin relative to 1% of viewport's smaller dimension
vmax relative to 1% of viewport's larger dimension
ch relative to width of the "0" (zero)
ex rarely relative to the x-height of the current font
