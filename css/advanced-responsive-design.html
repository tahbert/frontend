Media Queries
General A technique in CSS3 used to add breakpoint for different styles on different screens or devices
Mobile First Design makes the page display faster on smaller devices
 		see more
Creating Rules For mobile (all .col-) [class*="col-"] { /* smaller than 600px */
  width: 100%;
}
For tablets (.col-s-) @media only screen and (min-width: 600px) { /* from 600px to 767px */
  .col-s-1 {width: 8.33%;}
  .col-s-2 {width: 16.66%;}
  .col-s-3 {width: 25%;}
  .col-s-4 {width: 33.33%;}
  .col-s-5 {width: 41.66%;}
  .col-s-6 {width: 50%;}
  .col-s-7 {width: 58.33%;}
  .col-s-8 {width: 66.66%;}
  .col-s-9 {width: 75%;}
  .col-s-10 {width: 83.33%;}
  .col-s-11 {width: 91.66%;}
  .col-s-12 {width: 100%;}
}
For desktop (.col-) @media only screen and (min-width: 768px) { /* 768px or larger */
  .col-1 {width: 8.33%;}
  .col-2 {width: 16.66%;}
  .col-3 {width: 25%;}
  .col-4 {width: 33.33%;}

}
Typical Breakpoints Extra small devices (small phones) @media only screen and (max-width: 600px) {...}
Small devices (portrait tablets, large phones) @media only screen and (min-width: 600px) {...}
Medium devices (landscape tablets) @media only screen and (min-width: 768px) {...}
Large devices (laptops/desktops) @media only screen and (min-width: 992px) {...}
Extra large devices (large laptops and desktops) @media only screen and (min-width: 1200px) {...}
Orientation landscape|portrait
when the width is larger than the height and vice versa 		@media only screen and (orientation: landscape) {}
@media only screen and (orientation: portrait) {}
