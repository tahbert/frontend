Object-fit and Object-position property (for <img> and >video>)

Elements Explaination Visual Code
object-fit object-fit: fill|contain|cover|scale-down|none|initial|inherit;
Specify how an <img> or <video> should be resized to fit its container

Original image: width: 400px; height: 300px;
    With width: 200px; height: 250px;
  • fill: keep aspect ratio, stretch or squish if necessary
  • none: the image is not resized

object-fit: fill; (default)

object-fit: none;
    With width: 200px; height: 250px;
  • contain: keep aspect ratio, resized to fit
  • cover: keep aspect ratio, clipped to fit
  • scale-down: scale down to the smallest version of none or contain

object-fit: contain;

object-fit: cover;

object-fit: scale-down;
object-position object-position: position|initial|inherit;
Used with object-fit to specify how an <img> or <video> should be positioned with x/y coordinates inside its "own content box" (default: 50% 50%)
Negative value allowed

object-fit: cover; object-position: 0 0;
