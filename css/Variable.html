CSS variable
  • make the code easier to read (more understandable)
  • make it much easier to change the color values

global variable

can be used by the entire document, declared inside :root		 :root {--blue: #0000ff;}
Declaration

local variable

used only inside the selector where it is declared		 button {--blue: #0000ff;}
var() function var(--name)
var(--name, value)
insert the value of a CSS variable
  • value: the fallback value if the variable is not found
 div {color: var(--blue)}
div {color: var(--blue, blue)}
Overriding Variable local variable overrides global variable :root {--blue: #1e90ff;}
body {background: var(--blue)}
button {
 --blue: #0000ff;
 background: var(--blue); // #0000ff applied
}
JavaScript CSS variables have access to the DOM
=> possible to change them with JS

change the variable by JS
Media Queries possible to change variables with Media Queries :root {--blue: blue;}
&media (max-width: 768px) {
 :root {--blue: blue;}
