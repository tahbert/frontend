Specify the origin position of transformed elements

(default: 50% 50% 0 for 3D transfromation)
 required transfrom property 		transform-origin: 20% 40%;

perspective
(3D transform) 		perspective: none|length
Define how far the object is away from the user
lower value will result in a more intensive 3D effect
perspective: none;
perspective: 100px;
perspective: 400px;
perspective-origin
(3D transform) 		perspective-origin: x-axis y-axis|initial|inherit; (default: 50% 50%)
Define at from which position the user is looking at the 3D-positioned element
required perspective property
    Possible values:
  • position words
  • length
  • %
perspective-origin: left;
perspective-origin: bottom right;
perspective-origin: -90%;
backface-visibility
(3D transform) 		backface-visibility: visible|hidden|initial|inherit;
Define whether or not the back face of an element should be visible when facing the user
backface: mirror image of the front face
Useful: when an element is rotated. It lets you choose if the user should see the back face or not
Safari: -webkit- prefix
backface-visibility: hidden;
BACKFACE
backface-visibility: vision;

Elements Explaination Example
translate(x,y) Move an element from its current position

translate(50px,20px);
translateX(x)
normal
translateX(50px);
translateY(y)
normal

rotate(angle) Rotate an element clockwise or counter-clockwise
Possitive value: clockwise
Negative value: counter-clockwise
normal
rotate(45deg);
rotate(-90deg);
scale(x,y) Increase or decrease the width and height of an element
normal
scale(2,2);
scale(0.8,0.8);
scaleX(x) Increase or decrease the width of an element
normal
scaleX(2);
scaleX(0.8);
scaleY(y) Increase or decrease the height of an element
normal
scaleY(2);
scaleY(0.8);
skew(x,y) Skew an element along the X and Y-axis by the given angles
normal
skew(45deg,10deg);
skew(-45deg,10deg);

skewX(x) Skew an element along the X-axis by the given angles
normal

skewY(y) Skew an element along the Y-axis by the given angles.
normal
skewY(10deg);
skewY(-10deg);
