Animations

Elements Explaination Visual Code
General Allow animation without using JS or Flash!
animation: gradually change from one style to another, and back to original style when finishing
animation-name required to bind the animation to an element
@keyframes @keyframes animation-name {keyframes-selector {css-styles;}}
Specify the animation code
Note: The !important rule is ignored in a keyframe
? multiple available
keyframes-selector: from, to @keyframes example {
  from {background-color: green;}
  to {background-color: yellow;}
}
keyframes-selector: percentage
Useful for adding many style changes 		@keyframes example {
  0% {background-color: green;}
  25% {background-color: red;}
  50% {background-color: purple;}
  100% {background-color: yellow;}
}
animation
shorthand 		animation: name duration timing-function delay iteration-count direction fill-mode play-state;
(default: none 0 ease 0 1 normal none running)
animation: shorthand 5s linear 2s infinite alternate;
animation-name
 animation-name: none|text|initial|inherit;
Specify a name for the @keyframes animation
animation-duration
required 		animation-duration: time|initial|inherit; (default: 0)
Specify how long an animation should take to complete
animation-timing-function animation-timing-function: ease|linear|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end)|cubic-bezier(n,n,n,n)|initial|inherit;
Specify the speed curve of the animation
  • steps(int,start|end): intervals (positive integer greater than 0), second parameter is optional
  • cubic-bezier: values from 0 to 1
animation-timing-function: ease;
animation-timing-function: linear;
animation-timing-function: ease-in;
animation-timing-function: ease-out;
animation-timing-function: ease-in-out;
animation-timing-function: step-start;
animation-timing-function: step-end;
animation-timing-function: steps(2);
animation-timing-function: cubic-bezier(0,0,0.5,1);
animation-delay animation-delay: time|initial|inherit; (default: 0)
Specify a delay for the start of an animation
Negative value allowed
animation: delay0 5s linear 0s;
animation: delay1 5s linear 2s;
animation: delay1 5s linear -2s;
animation-iteration-count animation-iteration-count: number|infinite|initial|inherit; (default: 1)
Specify how many times an animation should run
animation-direction animation-direction: normal|reverse|alternate|alternate-reverse|initial|inherit;
Specify whether an animation should be played forwards, backwards or in alternate cycles
animation-fill-mode animation-fill-mode: none|forwards|backwards|both|initial|inherit;
Specify a style for the element when the animation is not playing (before it starts, after it ends, or both)
animation: fillmode-none 5s none;
animation: fillmode-forwards 5s forwards;
animation: fillmode-backwards 5s backwards;
animation: fillmode-both 5s both;
animation-play-state animation-play-state: running|paused|initial|inherit;
Specify whether the animation is running or paused
Note: Use this property in a JS to pause an animation in the middle of a cycle
.playstate {animation: playstate 5s infinite alternate;}
.playstate:hover {animation-play-state: paused;}
