Opacity Property

Elements Explaination Example
General Specify the opacity for an element (0.0 - 1.0)
Often with :hover to change the opacity on mouse-over
Problem Childrent inherit the same transparency => hard to read text
background: rgb(76, 175, 80); opacity: 0.5;
Solution Using RGBA
background: rgba(76, 175, 80, 0.5);
