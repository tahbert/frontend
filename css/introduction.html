General Cascading Style Sheets
Cascading: a style applied to a parent element will also apply to all children elements within the parent
Syntax
Three Ways External: most common
Using external file mystyle.css 		<link rel="stylesheet" href="mystyle.css">
Internal: unique style for one single document
Using <style></style> 		<head>
  <style>
    --- some code here ---
  </style>
</head>
Inline: unique style for a single element
Using stype attribute 		<p style="color:red;">This is a paragraph.</p>
