grid-template-areas grid-template-areas: none|itemnames
Specify areas within the grid layout
  • apostrophes (' ') define each row
  • space seperate the columns in each row
  • period sign represent a grid item with no name
 
        .grid-container {
          grid-template-areas:
            'myArea myArea . '
            ' . yourArea yourArea';
        }
        
        .item1 {grid-area: myArea;}
        .item4 {grid-area: yourArea;}

shorthand 		grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end | itemname;

grid-row
shorthand 		grid-row: grid-row-start / grid-row-end; (default: auto / auto)
Define on which row(s) to place an item
grid-column
shorthand 		grid-column: grid-column-start / grid-column-end; (default: auto / auto)
Define on which column(s) to place an item
Specific
grid-row-start grid-row-start: auto|row-line
Specify on which row-line the item will start
grid-column-start grid-column-start: auto|span n|column-line
Specify on which column-line the item will start, or the number of columns the item will span
grid-row-end grid-row-end: auto|span n|row-line
Specify on which row-line the item will end, or how many rows an item will span
grid-column-end grid-column-end: auto|span n|column-line
Specify on which column-line the item will end, or how many columns an item will span
