Combinator Selectors (select elements based on a specific relationship between them)

Elements Explaination Visual Code
element1 element2
.class1 .class2 		Select all the descendants of a specific element|class div p {
} /*select all <p> that are descendant <div>*/
.sidebar .left {
} /*select all .left that are descendant .sidebar*/
element1 > element2 Select all the children of a specific element div > p {
} /*select all <p> that are children of <div>*/
element1 ~ element2 Select all siblings after a specific element. div ~ p {
} /*select all <p> that is siblings of and next to <div>*/
element1 + element2 Select only one sibling immediately after a specific element div + p {
} /*select the first <p> that is sibling of and immediately after <div>*/

Attribute Selectors (select elements based on an attribute or attribute value)

Elements Explaination Visual Code
[attribute] Select all elements with a specific attribute [target]

<a href="http://www.disney.com" target="_blank">disney.com</a>
<a href="http://www.wikipedia.org" target="_top">wikipedia.org</a>
[attribute=value] Select all elements with a specific attribute and its value [target=_blank]

<a href="http://www.disney.com" target="_blank">disney.com</a>
<a href="http://www.wikipedia.org" target="_top">wikipedia.org</a>
[attribute|=value] Select all elements with a specific attribute and its value
starting with a specified word
(Required: the value has to be a whole word,either alone, or followed by a hyphen) 		[class|=test]

<div class="test">The third div element.</div>
<div class="test-ex">The fourth div element.</div>
<div class="test_ex">The fourth div element.</div>
<div class="testex">The fourth div element.</div>
[attribute^=value] Select all elements with a specific attribute and its value
starting with a specified string
 [class^=test]

<div class="test">The third div element.</div>
<div class="test-ex">The fourth div element.</div>
<div class="test_ex">The fourth div element.</div>
<div class="testex">The fourth div element.</div>
[attribute$=value] Select all elements with a specific attribute and its value
ending with a specified string
 div[class$=test]

<div class="first-test">The first div element.</div>
<div class="second_test">The second div element.</div>
<div class="third test">The third div element.</div>
<div class="lasttest">The third div element.</div>
[attribute~=value] Select all elements with a specific attribute and its value
containing an exact word
 div[class~=test]

<div class="first-test">The first div element.</div>
<div class="second_test">The second div element.</div>
<div class="third test">The third div element.</div>
<div class="lasttest">The third div element.</div>
[attribute*=value] Select all elements with a specific attribute and its value
containing a specified string
 div[class*=test]

<div class="test_ex">The first div element.</div>
<div class="second_test">The second div element.</div>
<div class="third test">The third div element.</div>
<div class="lasttest">The third div element.</div>
