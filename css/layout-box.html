outline-offset outline-offset: length|initial|inherit; (Default: 0 | transparent)
Add space between an outline and the edge or border
    Differ from borders in three ways
  • An outline is a line drawn around elements, outside the border edge
  • An outline does not take up space
  • An outline may be non-rectangular

Problem Actual width of an element = width + padding + border
Actual height of an element = height + padding + border

Solution box-sizing: content-box|border-box|initial|inherit;
Set box-sizing: border-box; to inclue paddings, borders to the total width|height.
