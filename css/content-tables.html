Table Layout
table-layout table-layout: auto|fixed|initial|inherit;
Specify the algorithm used to lay out table cells, rows, and columns
Added to <table>
  • auto: The column width is set by the widest unbreakable content in the cells. The content will dictate the layout
  • fixed: The table and column widths are set by the widths of table and col or by the width of the first row of cells. Cells in other rows do not affect column widths. If no widths are present on the first row, the column widths are divided equally across the table, regardless of content inside the cells.
    Users will see the top of the table while the browser loads and renders rest of the table => quicker!
