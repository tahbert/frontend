<meta> The document metadata

<meta http-equiv="value"> 		content-security-policy: Specify a content policy for the document

<meta http-equiv="refresh" content="300">
name attribute: Specify the name for the metadata
<meta name="value"> 		author: Specify the name of the author of the document
<meta name="author" content="John Doe">

description: Specify Specifies a description of the page (Search engines can pick up this description to show with the results of searches)
<meta name="description" content="Free web tutorials">

generator: Specify one of the software packages used to generate the document
<meta name="generator" content="FrontPage 4.0">

keywords always: Specify a comma-separated list of keywords - relevant to the page (Informs search engines what the page is about)
<meta name="keywords" content="HTML, meta tag, tag reference">

viewport always: Specify the viewport
<meta name="viewport" content="width=device-width, initial-scale=1.0">

application-name: Specify the name of the Web application
