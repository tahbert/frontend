Elements Explaination Example
Video
<video></video> Supported formats: MP4, WebM, Ogg
The text between the tags: For browsers that do not support
Optional Attributes
controls <video controls>
Specify that video controls should be displayed.
autoplay <video autoplay>
Specify that the video will automatically start playing
Chromium do not allow autoplay. However, autoplay muted is always allowed
muted <video muted>
Specify that the audio output of the video should be muted.
height|width always <video height|width="pixels">
Specify the height|width of a video player, in pixels
loop <video loop>
Specify that the video will start over again, every time it is finished
poster <video poster="URL">
Specify an image to be shown while the video is downloading, or until the user hits the play button
If this is not included, the first frame of the video will be used instead
preload <video preload="auto|metadata|none">
Specify if and how the author thinks that the video should be loaded when the page loads.
(load the entire video | load only metadata | NOT load the video)
Ignored if autoplay is set
src <video src="URL">
Specify the location (URL) of the video file
Using <source> for sure on all browsers support

Tracks
<track></track> Specify text tracks for <audio> or <video> elements
(subtitles, caption files or other files containing text, that should be visible when the media is playing)
Supported format: .vtt
Optional Attributes
src <track src="URL">
Specify the URL of the track file 		<track src="subtitles_en.vtt" kind="subtitles" srclang="en">
default <track src="URL default">
Specify that the track is to be enabled if the user's preferences do not indicate that another track would be more appropriate
Only be one <track> element with a default attribute per <media> 		<track src="fgsubtitles_en.vtt" kind="subtitles" srclang="en" label="English" default>
<track src="fgsubtitles_no.vtt" kind="subtitles" srclang="no" label="Norwegian">
kind <track kind="value">
Specify the kind of text track.
  • captions: translation of dialogue and sound effects (suitable for deaf users)
  • chapters: chapter titles (suitable for navigating the media resource)
  • descriptions: a textual description of the video content (suitable for blind users)
  • metadata: content used by scripts. Not visible for the user
  • subtitles: used to display subtitles in a video
 <track src="subtitles_en.vtt" kind="subtitles" srclang="en">
srclang <track srclang="language_code">
required if kind="subtitles". Specify the language of the track text data
 <track src="subtitles_en.vtt" kind="subtitles" srclang="en">
label <track lable="text">
Specify the title of the text track, also for the browser to list available tracks		 <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English">
Source
<source></source> Specify multiple media resources for media elements, such as <video>, <audio>, <picture>
The first recognized is used
media <source media="media_query">
Accept any valid media query that would normally be defined in a CSS
Multiple value are allowed with Operators: and|not|, 		see all values here
src <source src="URL">
Specify the URL of the media file
type <source type="MIME-type">
Specify the Internet media type (formerly known as MIME type) of the media resource.
 <source src="horse.ogg" type="audio/ogg">
