Attributes href attribute important Specify the URL the link goes to.
<a target="_blank|_self|_parent|_top|framename">
target attribute: Specify where to open the linked document
_self in the same frame as it was clicked (default)
_blank in a new window or tab
_parent in the parent frame
_top in the full body of the window
framename in the named iframe
Other attributes: download, hreflang, media, ping, referrerpolicy, rel, type see more
