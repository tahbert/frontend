Frame Elements

Frame Elements

Elements Explaination Example
<iframe></iframe> <iframe src="url" title="description"></iframe>
Specify an inline frame, especially another web page
refuse to connect error
Attributes
src always Specify the URL of the document to embed in an iframe
title always For screen readers to read out what the content of the frame is
name <iframe name="text">
Specify the name of an <iframe>
Can be used to reference the element in a JS
Or as the value of the target attribute of an <a> or <form>
Or the formtarget attribute of an <input> <button>
height|width Default: height: 150 pixel | width: 300 pixel
referrerpolicy <iframe referrerpolicy="value"> see values
Specify which referrer information to send when fetching the iframe
allow <iframe allow>
Specify a feature policy for the <iframe>
allowfullscreen <iframe allowfullscreen="true|false">
Set to true if the <iframe> can activate fullscreen mode by calling the requestFullscreen() method
allowpaymentrequest <iframe allowpaymentrequest="true|false">
Set to true if a cross-origin <iframe> should be allowed to invoke the Payment Request API
loading <iframe loading="eager|lazy">
Specify whether a browser should load an iframe immediately or to defer loading of iframes until some conditions are met
sandbox <iframe sandbox="value"> see values
Enable an extra set of restrictions for the content in the iframe
srcdoc <iframe srcdoc="HTML_code">
Specify the HTML content of the page to show in the inline frame
Expected to be used together with the sandbox and seamless attributes
Override the content specified in the src attribute (if present).
