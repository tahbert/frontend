Elements Explaination Example
<img> Define an image
<map></map> Define a client-side image map (one with clickable areas)
name attribute realated to usemap attribute
<canvas></canvas> A container for drawing graphics via JS complete tutorials
id attribute: always To be referred to in a script)
width, height attribute: always Define the size of canvas
 		Your browser does not support the HTML canvas tag
<svg></svg> A container for SVG graphics (vector-based graphics in XML format) complete tutorials
(SVG stands for Scalable Vector Graphics
SVG is used to define graphics for the Web
SVG is a W3C recommendation)
<figure></figure> Specify self-contained content, like illustrations, diagrams, photos, code listings, etc.
the content is related to the main flow
the position is independent of the main flow
=> Not affect the flow of the document when removed.
<figcaption></figcaption>: Caption for the <figure>
<picture> Display different pictures for different devices or screen sizes
Why 1: Bandwidth - It's not necessary to load a large image on a small screen
Why 2: Format Support - Some browsers, devices may not support all image formats

Attributes for <img>

Elements Explaination Example
src required <img src="URL">
Specify the path to the image
It is almost always best to use relative URLs
alt required <img alt="text">
Specify an alternate text for the image
- To create a tooltip for an image, use the title attribute!
- The text should explain where the link goes if inside an <a>
- Use alt="" if the image is only for decoration
width, height always <img width|height="pixels">
Specify width and height for the image
usemap <img usemap="#mapname">
Specify an image as a client-side image map
Can't be used if <img> is a descendant of an <a> or <button>
crossorigin <img crossorigin="anonymous|use-credentials">
Allow images from third-party sites that allow cross-origin access to be used with canvas
ismap <img ismap>
Specify that the image is part of a server-side image map
loading <img loading="eager|lazy"> (default: eager)
Specify how a browser should load an image.
  • eager: Loads an image immediately
  • lazy: Defer loading of images until some conditions are met
    (only to images which are positioned below the fold)
referrerpolicy <img referrerpolicy="value"> see more
Specify which referrer information to use when fetching an image
sizes <img sizes="sizes">
Specify image sizes for different page layouts
srcset <img srcset="URL-list">
Specify a list of image files to use in different situations

<area> Elements for <map>

Elements Explaination Example
<area> Define an area inside an image map
Attributes for <area>
shape required <area shape="default|rect|circle|poly">
Specify the shape of an area
  • default: define the entire region
  • rect: a rectangular region
  • circle: define a circular region
  • poly: define a polygonal region
coords <area coords="value">
Specify the coordinates of the area: x-axis and y-axis
  • default: no need for this attribute
  • rect: x1,y1,x2,y2
  • circle: x,y,radius
  • poly: x1,y1,x2,y2,..,xn,yn
download <area download="filename">
Specify that the target will be downloaded when the hyperlink is clicked
(Default: the original filename is used) 		<area shape="rect" coords="0,0,82,126" alt="Sun" href="info_about_the_sun.htm" download="sun">
referrerpolicy <area referrerpolicy="value"> see more
Specify which referrer information to send when the hyperlink is clicked 		<area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun" referrerpolicy="same-origin">
target <area rel="_blank|_self|_parent|_top|framename">
Specify where to open the target URL
  • _blank: in a new window or tab
  • _self: in the same frame as it was clicked
  • _parent: in the parent frame
  • _top: in the full body of the window
  • framename: in a named iframe
href advisory <area href="URL">
Specify the target URL. 		<area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun" target="_blank">
alt <area alt="text">
Required if the href attribute is set 		<area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun" target="_blank">
media advisory <area media="value"> see more
Specify what media/device the target URL is optimized for
Only used if the href attribute is set 		<area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" media="screen and (min-color-index:256)">
type advisory <area rel="media_type"> see more
Specify the Internet media type of the target URL
Only used if the href attribute is set 		<area shape="rect" coords="0,0,82,126" alt="Sun" href="/images/sun.gif" type="image/gif">>
hreflang <area hreflang="language_code"> see more
Specify the language of the target URL
Only used if the href attribute is set 		<area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" hreflang="en">
rel <area rel="value"> see more
Relationship between the current and linked document
Only used if the href attribute is set 		<area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" rel="alternate">
