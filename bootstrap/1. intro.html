General the most popular HTML, CSS, JS framework for creating responsive, mobile-first websites
The main differences between Bootstrap 5 and Bootstrap 3 & 4: Bootstrap 5 has switched to JS instead of jQuery.
Where to get 1. Include Bootstrap 5 from a CDN (advantage: faster loading time) <!-- Latest compiled and minified CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/css/bootstrap.min.css" rel="stylesheet">
<!-- Latest compiled JS (if you just use the CSS part of Bootstrap, you don't need this) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/js/bootstrap.bundle.min.js"></script>
2. Download Bootstrap 5 from getbootstrap.com
