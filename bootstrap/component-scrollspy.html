Component - Scrollspy
Elements Explaination Visual Code
Definition Automatically update links in a navigation list based on scroll position
Basic data-bs-spy="scroll" to the scrollable area element (often <body>)
required relative position to work properly
data-bs-target="#id-name|.class-name" for the navbar is connected with the scrollable area
data-bs-offset specify offset from top (default: 10px)
Nested nav If a nested .nav is .active, its parents will also be .active
