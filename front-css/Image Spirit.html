Image Sprites

Elements Explaination Example
General A collection of images put into a single image
Useful: reduce the number of server requests and save bandwidth
Basic Example
  • <img> define a transparent image only because the src attribute cannot be empty
  • background: url(imgs/img_sprites.gif) -91px 0; define the background image and its position
  • width: 43px; height: 44px; define the portion of the image to use
Navigation List

  • .navlist li {position:absolute;} absolute positioned, enable us position <li> relative to <ul>
  • .navlist li {margin:0;padding:0;list-style:none;} resetting the default
  • .navlist a {display: block;}
