Shadow

Elements Explaination Example
text-shadow text-shadow: h-shadow v-shadow blur-radius color|none|initial|inherit;
required h-shadow, v-shadow
Multiple: seperated by commas

text-shadow: 2px 2px;

text-shadow: 2px 2px green;

text-shadow: 2px 2px 5px green (with 5px blur);

text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF; (multiple)
box-shadow box-shadow: none|h-offset v-offset blur spread color|inset|initial|inherit;
required h-offset, v-offset
Multiple: seperated by commas
box-shadow: 10px 10px;
box-shadow: 10px 10px grey;
box-shadow: 10px 10px 5px grey;
box-shadow: 10px 10px 5px 10px grey;
box-shadow: 10px 10px 5px -10px grey;
box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2),
0 6px 20px 0 rgba(0, 0, 0, 0.19); (multiple)
spread optional value: increase or decrease the size of shadow
inset optional value: change the shadow from an outer shadow (outset) to an inner shadow
Best Practice
  • Lit from the same global light source.
  • Same ratio between horizontal and vertical offsets
  • Closer to the user:
    the offset should increase,
    the blur radius should increase,
    the opacity should decrease.
  • Match background color
