The background Property

The background property is a shorthand property for specifying all the background properties in one declaration.

background: #ffffff url("imgs/tree.png") no-repeat right top;
When using background-size in the shorthand, position required

background: url(imgs/tahbert.png) top left/contain no-repeat;
background-color background-color: color|transparent|initial|inherit; (default: transparent)
Set the background color for an element
(The background includes padding, border) 		background-color: lightgreen;
Transparent using opaticy property(Not good: apply to all children) background-color: lightgreen; opacity: 0.3;
Transparent using RGBA (better way) background-color: rgba(0, 128, 0, 0.3);
background-image background-image: url|none|initial|inherit; (default: none)
Set one or more background images for an element
(Default: top-left, repeat vertically and horizontally)
(Always set a background-color in case the image is unavailable.) 		background-image: url("imgs/bg-pattern.png");
background-color: lightgrey;
Gradient Background see more background-image: linear-gradient(green, yellow);
background-repeat background-repeat: repeat|repeat-x|repeat-y|no-repeat|space|round|initial|inherit; (default: repeat)
Specify how the background image repeat play it 		background-image: url("imgs/flower-tree.png");
background-repeat: no-repeat;
background-attachment background-attachment: scroll|fixed|local|initial|inherit; (default: scroll)
Specify whether the background image should scroll or be fixed
background-position background-position: value|initial|inherit; (default: top left)
Specify the position of the background image
Value 1: use the word top, right, bottom, left, center
(If one keyword, the other value will be center) 		background-position: center;
background-position: top; (=> top center)
background-position: top right;
Value 2: Use x% y%
(If one value, the other value will be50%) 		background-position: 30%; (=> 30% 50%)
background-position: 30% 50%;s
Value 3: Use xpos ypos
(If one value, the other value will be50%)
(Units can be pixel or other CSS units) 		background-position: 30px; (=> 30px 50%)
background-position: 30px 50px;
background-position: top 30px right 50px;
background-origin background-origin: padding-box|border-box|content-box|initial|inherit; (default: padding-box)
Specify the origin position of a background image
(background-attachment: fixed; => no effect)
padding-box: top left corner of the padding edge
border-box: top left corner of the border
content-box: top left corner of the content
background-clip background-clip: border-box|padding-box|content-box|initial|inherit; (default: border-box)
Define how far the background should extend within an element
border-box: behind the border
padding-box: inside edge of the border
content-box: to the edge of the content box
background-size background-size: auto|length|cover|contain|initial|inherit; (default: auto)
Specify the size of the background image
auto: original size
cover: cover the entire container
contain: fully visible inside the container
length with 1 value: set the width, height: auto
length with 2 values: set the width and height
background-blend-mode background-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity;
Define the blending mode of each background layer (color and/or image) play it
