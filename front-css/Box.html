Box Outline

Elements Explaination Example
outline shorthand outline: width style color|initial|inherit;
  • default: medium invert color
  • not a part of the element's dimensions => may overlap other content
outline: 5px dashed green
outline-width
outline-style
outline-color
 similar to those of border
outline-offset outline-offset: length|initial|inherit; (default: 0 | transparent) add space between an outline and the edge or border
    differ from borders in 3 ways
  • An outline is a line drawn around elements, outside the border edge
  • An outline does not take up space
  • An outline may be non-rectangular
outline-offset: 4px;

Box Sizing

Elements Explaination Example
Problem actual width = width + padding + border
actual height = height + padding + border
=> adding padding and border makes the box bigger.
Solution box-sizing: content-box|border-box|initial|inherit;
set border-box; to inclue paddings, borders to the total width|height

good practice to apply to universal selector

                                * {
                                    box-sizing: border-box;
                                }
