CSS Transitions

Elements Explaination Example
transition
shorthand 		transition: property duration timing-function delay|initial|inherit; (default: all 0s ease 0s)
Change property values smoothly, over a given duration
Multiple: seperated by commas
transition: width 2s ease-out 1s,
height 2s ease-in 0.5s,
background-color 0 ease 0;
Transition + Transformation
transition: width 2s,
height 2s,
background-color 2s;
transform 2s;
transition-property transition-property: all|none|property|initial|inherit;
Specify the property name the transition effect is for
Multiple: seperated by commas
Typically occur when a user hover over an element
transition-property: width, height;
transition-duration transition-duration: time|initial|inherit; (default: 0s)
Specify the duration the transition takes to complete
seconds (s) or milliseconds (ms)
transition-duration: 2s;
transition-timing-function transition-timing-function: ease|linear|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end)|cubic-bezier(n,n,n,n)|initial|inherit;
Specify the speed curve
  • steps(int,start|end): intervals (positive integer greater than 0), second parameter is optional
  • cubic-bezier: values from 0 to 1
transition-timing-function: ease;
transition-timing-function: linear;
transition-timing-function: ease-in;
transition-timing-function: ease-out;
transition-timing-function: ease-in-out;
transition-timing-function: step-start;
transition-timing-function: step-end;
transition-timing-function: steps(2);
transition-timing-function: cubic-bezier(0,0,0.5,1);
transition-delay transition-delay: time|initial|inherit; (default: 0s)
Specify the duration the transition takes to start
seconds (s) or milliseconds (ms)
transition-delay: 1s;
